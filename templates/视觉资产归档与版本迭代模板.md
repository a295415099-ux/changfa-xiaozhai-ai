# 视觉资产归档与版本迭代模板

## 文件夹结构

把素材放进对应工作区，目录固定为：

```text
assets/ecommerce/首图/{产品}/{平台}/{版本}/
assets/ecommerce/商详页/{产品}/{平台}/{版本}/
assets/ecommerce/首页/{产品或店铺}/{平台}/{版本}/
```

示例：

```text
assets/ecommerce/商详页/虫草首乌发膜/天猫/V03_0717上线版/
```

版本文件夹内可以放 JPG、JPEG、PNG、WebP、GIF、SVG、MP4、MOV、WebM 或 M4V。

## 版本说明

在版本文件夹内复制一份 `_版本说明.json`：

```json
{
  "project": "AI项目04",
  "product": "虫草首乌发膜",
  "platform": "天猫",
  "version": "V03 0717上线版",
  "date": "2026-07-17",
  "status": "上线测试",
  "note": "本版重点调整首屏利益点和产品信任证据。"
}
```

## 数据记录

在同一版本文件夹内放 `_数据.csv`：

```csv
指标,上线前,上线后,变化,结论
点击率,,,,
跳失率,,,,
收藏加购率,,,,
转化率,,,,
```

重新运行总控台内容构建后，素材和数据会自动进入对应工作区。
