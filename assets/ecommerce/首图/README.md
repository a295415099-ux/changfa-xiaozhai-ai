# 首图视觉资产库

这里统一归档商品主图、搜索首图、货架图和广告首图。总控台顶部点击 `首图`，即可按产品、平台和版本浏览。

## 文件夹结构

```text
assets/ecommerce/首图/{产品}/{平台}/{版本}/
```

例如：

```text
assets/ecommerce/首图/虫草首乌发膜/天猫/V03_0717上线版/
```

版本建议按 `V序号_日期_阶段` 命名，例如 `V01_0627初稿`、`V02_0703改版`、`V03_0717上线版`。

## 每个版本放什么

- 必需：该版本的图片或视频文件。
- 可选：`_版本说明.json`，记录负责人、目标、改动和状态。
- 可选：`_数据.csv`，记录上线前、上线后和变化值。

图片支持 `jpg`、`jpeg`、`png`、`webp`、`gif`、`svg`，视频支持 `mp4`、`mov`、`webm`、`m4v`。

## 建议复盘指标

- 曝光量、点击量、点击率。
- 搜索进店、商品访问、加购和成交。
- 同一产品不同版本的 CTR、转化率与投放效率变化。

历史项目素材已经通过 `assets/ecommerce/视觉资产索引.json` 接入，不需要立刻搬动；后续新增版本按上述文件夹结构放入即可。
