# 平台首页视觉资产库

这里统一归档天猫、淘宝、京东、抖音等平台的店铺首页、品牌首页和活动首页。总控台顶部点击 `平台首页`，即可按店铺或产品线、平台和版本浏览。

## 文件夹结构

```text
assets/ecommerce/首页/{产品线或店铺}/{平台}/{版本}/
```

例如：

```text
assets/ecommerce/首页/长发小寨旗舰店/天猫/V03_0717上线版/
```

版本建议按 `V序号_日期_阶段` 命名，例如 `V01_0627明星版`、`V02_0703视觉文案改版`、`V03_0717上线版`。

## 每个版本放什么

- 必需：首页完整长图，或按页面顺序编号的模块图。
- 可选：`_版本说明.json`，记录平台、目标、主推产品和改动。
- 可选：`_数据.csv`，记录上线前、上线后和变化值。

图片支持 `jpg`、`jpeg`、`png`、`webp`、`gif`、`svg`，视频支持 `mp4`、`mov`、`webm`、`m4v`。不同平台请分开建文件夹，避免同名版本混在一起。

## 建议复盘指标

- 首页访客、跳失率、平均停留和核心模块点击。
- 主推 SKU 导流、收藏加购和成交转化。
- 同一平台不同版本，以及不同平台首页的承接效率变化。

历史项目素材已经通过 `assets/ecommerce/视觉资产索引.json` 接入，不需要立刻搬动；后续新增版本按上述文件夹结构放入即可。
