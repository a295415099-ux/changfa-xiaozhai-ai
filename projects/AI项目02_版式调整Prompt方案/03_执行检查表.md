## 5. 版本沉淀规则

每次使用 AI项目02 产出版式调整方案后，都要记录：

| 字段 | 内容 |
|---|---|
| 项目编号 | AI项目02 |
| 使用对象 | 例如商详页首屏、首页首屏、小红书长图、包装正面 |
| 原始版本 | 初稿 / 打样稿 / 正稿 / 上线稿 |
| 调整目标 | 提升点击、提升理解、增强高级感、统一风格等 |
| 使用 Prompt | 粘贴最终使用版本 |
| 输出版本 | AI 生成稿 / 设计师改稿 / 上线稿 |
| 复核结论 | 通过 / 待修改 / 废弃 |
| 后续动作 | 继续优化、上线测试、归档模板 |
