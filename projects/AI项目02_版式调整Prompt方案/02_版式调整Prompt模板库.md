# 版式调整 Prompt 模板库

## 1. 商详页首屏版式调整 Prompt

```text
请帮我优化这张淘宝商详页首屏版式。

目标：让手机端用户 3 秒内看懂产品适合谁、解决什么问题、凭什么可信。

必须保留：
品牌名、产品图、核心功效、适用人群、关键证据、规格或活动信息。

调整要求：
1. 首屏只突出一个主卖点，主标题放大并前置。
2. 产品图必须成为视觉中心，不要被文字遮挡。
3. 适用人群用 2-3 个短标签表达，不要长句堆叠。
4. 功效证据用“结果 + 证据来源”的方式呈现。
5. 保持长发小寨东方植萃、高级、清爽、可信的品牌感。
6. 手机端字号要足够清晰，一屏只承载一个核心信息。

禁止：
不要新增未经确认的医疗化功效；不要使用廉价促销风；不要让画面过满。

请输出：
版式问题诊断、调整后模块顺序、设计师改稿说明、AI 改图 Prompt。
```

## 2. 店铺首页版式调整 Prompt

```text
请帮我优化这张长发小寨店铺首页版式。

目标：让用户进入首页后快速知道品牌是谁、主推产品是什么、下一步应该点击哪里。

必须保留：
品牌名、主推产品、核心活动入口、明星/达人/证据素材、导航入口。

调整要求：
1. 首页首屏建立“品牌定位 + 主推 SKU + 行动入口”三层结构。
2. 模块顺序按购买路径排列：品牌信任、爆品推荐、场景分类、活动权益、复购入口。
3. 主推 SKU 视觉权重要高于普通模块。
4. 减少装饰性文字，保留可点击、可转化的信息。
5. 统一图片风格、字体层级和按钮样式。

禁止：
不要把首页做成单纯海报墙；不要让多个活动同时抢主视觉。

请输出：
新版首页模块结构、首屏版式建议、视觉层级说明、可交给设计师的改稿清单。
```

## 3. 主图 / 首图版式调整 Prompt

```text
请帮我优化这张电商主图版式。

目标：提升搜索货架中的点击效率，让用户一眼看懂产品、功效、人群和利益点。

必须保留：
产品图、品牌名、核心功效、规格、已确认利益点。

调整要求：
1. 产品图占据主要视觉面积，轮廓清晰。
2. 主卖点不超过 12 个字，放在最容易看到的位置。
3. 人群或场景标签控制在 2-3 个。
4. 背景保持干净，避免复杂纹理影响缩略图识别。
5. 色彩与长发小寨品牌系统保持一致。

禁止：
不要堆过多卖点；不要添加无法验证的极限词；不要让文字贴边。

请输出：
主图问题诊断、点击优化方向、3 个不同版本的版式 Prompt。
```

## 4. 小红书长图版式调整 Prompt

```text
请帮我优化这张小红书长图版式。

目标：增强停留、收藏和种草效率，让内容更像真实经验分享，同时保持品牌专业感。

必须保留：
标题、核心经验点、产品图、关键对比或步骤、品牌相关信息。

调整要求：
1. 标题要有强场景感，第一屏形成停留理由。
2. 内容按“痛点 - 方法 - 结果 - 产品承接”排列。
3. 每屏只讲一个重点，避免信息密度过高。
4. 用真实体验感表达功效，不要像硬广海报。
5. 保持东方长发、清爽、可信的视觉氛围。

禁止：
不要使用过度促销话术；不要堆满贴纸和装饰元素。

请输出：
长图分屏结构、每屏标题建议、版式调整 Prompt、发布前检查清单。
```

## 5. 包装正面版式调整 Prompt

```text
请帮我优化这款长发小寨包装正面版式。

目标：提升货架识别、系列区分和品牌高级感。

必须保留：
品牌名、产品名、核心功效、容量、必要法规信息、现有核心视觉符号。

调整要求：
1. 品牌名和产品名层级清楚，远看也能识别。
2. 功效信息简洁，不要把包装正面做成详情页。
3. 用颜色或图形区分头皮线、发丝线、发尾线、特护线。
4. 保留东方植萃和红瑶淘米水的品牌联想。
5. 正面留白充足，侧面承接成分、步骤和证据信息。

禁止：
不要使用过多小字；不要新增未经确认的功效宣称；不要破坏系列统一性。

请输出：
包装正面层级建议、系列化区分方案、AI 改图 Prompt、设计复核标准。
```

## 6. 汇报单页版式调整 Prompt

```text
请帮我优化这页老板汇报版式。

目标：让决策人 10 秒内看到结论、问题、动作和责任。

必须保留：
标题、核心结论、关键数据、行动计划、负责人或时间节点。

调整要求：
1. 结论前置，不要先铺背景。
2. 页面分成“结论 / 依据 / 动作 / 下一步”四块。
3. 数据用表格或重点数字呈现，不要大段文字。
4. 行动项要能直接分配责任和时间。
5. 风格参考长发小寨史总汇报：清晰、克制、有高级感。

禁止：
不要做成花哨海报；不要让装饰元素抢结论。

请输出：
新版信息结构、版式调整说明、可复制的 Prompt、汇报前检查清单。
```
