# WIN视角内容分层框架

## 概述

WIN视角是将内容按照其在蓄水链路中的角色分为三个层级，帮助团队看清每一条内容分别承担什么任务。

## 三层定义

### W：广触达（Wide Reach）

**它负责什么**：让更多目标人群先看到、先知道、先被触发

**课堂解释**：适合前置种草、打认知、扩大触达面

**典型内容形态**：
- 开箱 / 测评类短视频
- 话题性种草笔记
- 品牌故事 / 理念内容
- 达人合作种草内容
- 场景化兴趣激发内容

**判断标准**：这条内容的主要目的是不是让更多人"先知道"？

---

### I：深互动（In-depth Interaction）

**它负责什么**：让用户愿意评论、收藏、交流、形成更深层次兴趣

**课堂解释**：适合建立共鸣、强化场景、沉淀互动信号

**典型内容形态**：
- 教程 / 攻略 / How-to 内容
- 痛点共鸣类内容
- 问答 / 投票 / 互动话题
- 用户故事 / UGC 征集
- 场景深度演绎内容

**判断标准**：这条内容是不是让用户愿意停下来、参与进来、产生兴趣信号？

---

### N：向搜索与承接推进（Navigate to Conversion）

**它负责什么**：让用户进入搜索、商品页、店铺页、直播间或私域

**课堂解释**：真正让内容开始为后面的收割和转化铺路

**典型内容形态**：
- 搜索词种草内容（引导搜索特定词）
- 商品对比 / 选购指南
- 直播间预热引流内容
- 店铺页关联推荐内容
- 私域引流内容

**判断标准**：这条内容是不是在引导用户完成一个可追踪的下一步动作？

---

## 核心原则

- 不同层的内容判断标准不同，不能混着看
- 一个完整的内容蓄水体系三层都要有
- W层打基础 → I层深互动 → N层推向成交
- 从老板认知过渡到团队执行的桥梁就是WIN分层
