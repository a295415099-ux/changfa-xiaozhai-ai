# AI提示词模板库

以下是各模块可直接使用的AI提示词模板。使用时将"以下内容"替换为用户实际提供的内容列表/信息。

---

## 模块1：WIN分层判断

**提示词**：
```
请基于以下内容计划，帮我判断这些内容分别更偏广触达、深互动，还是往搜索和承接推进，并说明原因。
```

**建议补充信息**：内容清单、平台、当前阶段目标

**预期输出**：WIN视角内容分层表

---

## 模块2：购买理由检查

**提示词**：
```
请判断以下内容是否提前种下了"为什么需要它"的购买理由，而不是只在讲产品本身。请逐条给出判断和理由。
```

**建议补充信息**：标题、文案、脚本、产品信息

**预期输出**：购买理由种植检查表

---

## 模块3：搜索心智检查

**提示词**：
```
请判断以下内容是否在用户脑子里留下一个词、一个场景、一个判断，帮助用户后续形成搜索心智。请逐条分析每条内容留下了什么"搜索记忆点"。
```

**建议补充信息**：内容样本、关键词、场景词、卖点表达

**预期输出**：搜索心智种植检查表

---

## 模块4：收割导向判断

**提示词**：
```
请判断以下内容最终更适合导向什么收割动作：导搜索、导商品页、导店、导直播、导私域，并说明原因。
```

**建议补充信息**：内容方向、承接路径、经营目标

**预期输出**：收割动作服务表

---

## 模块5：时间匹配判断

**提示词**：
```
请根据[大促节点]节点节奏、直播排期和搜索优化时间，判断以下内容发布时间是不是合适，哪些需要前置，哪些可以后置。请说明调整建议。
```

**建议补充信息**：发布时间、节点节奏、直播排期、活动计划

**预期输出**：时间匹配检查表

---

## 模块6：热闹内容识别

**提示词**：
```
请帮我识别以下内容里，哪些只是看起来很忙、有热度，但没有为后续搜索和转化铺路。请逐条分析并说明理由。
```

**建议补充信息**：当前内容动作清单、数据表现、链路路径

**预期输出**：热闹内容识别表

---

## 模块7：真蓄水内容识别

**提示词**：
```
请帮我判断以下内容里，哪些属于真正的蓄水内容，并说明它们具体在为后面的哪个动作铺路（导搜索/导商品页/导店/导直播/导私域）。
```

**建议补充信息**：内容样本、承接路径、时间节奏

**预期输出**：真蓄水内容表 + 蓄水目标表

---

## 模块8：首轮蓄水动作建议

**提示词**：
```
请基于以上蓄水判断结果，为我输出首轮动作建议：先发什么、先在哪个平台种、先铺什么词、先导向哪里。请说明优先级顺序和理由。
```

**建议补充信息**：优先内容、节点时间、平台资源

**预期输出**：首轮蓄水动作表

---

## 模块9：团队动作拆解

**提示词**：
```
请把以上内容蓄水判断结果拆成内容团队、电商团队、直播团队、私域团队各自要执行的动作。每个团队列出具体要做的事情、时间节点和交付标准。
```

**建议补充信息**：蓄水判断表、团队分工、节点节奏

**预期输出**：团队动作分工表
