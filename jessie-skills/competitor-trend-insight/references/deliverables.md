# 交付物模板说明

做完AI三维洞察后，需要沉淀以下5份交付物。

---

## 1. 竞品传播分析表

**用途**：看清竞品在讲什么、推什么、怎么说

**建议字段**：
| 字段 | 说明 |
|------|------|
| 品牌 | 竞品品牌名称 |
| 平台 | 内容发布平台 |
| 主推产品 | 竞品重点在推的产品 |
| 核心卖点 | 竞品如何包装卖点 |
| 标题风格 | 标题的套路类型（数字/问答/痛点/结果型等） |
| 正文结构 | 内容组织方式（如：痛点→方案→产品→信任→下单） |
| 高频话术 | 竞品反复使用的关键词和句式 |
| 情绪调性 | 内容传递的情绪氛围 |
| 互动数据 | 点赞/收藏/评论/转发情况 |
| 可借鉴点 | 值得学习的策略或话术 |

---

## 2. 同行场景机会表

**用途**：看同类品牌都在打哪些场景，找自己的场景切入点

**建议字段**：
| 字段 | 说明 |
|------|------|
| 品牌 | 同行品牌名称 |
| 场景类型 | 使用场景/送礼场景/情绪场景/身份场景等 |
| 表达方式 | 该场景下怎么呈现（图文、视频、口播等） |
| 互动情况 | 该场景内容的互动数据 |
| 可借鉴点 | 值得学习的地方 |
| 差异化机会 | 可以做不同或更好的地方 |

---

## 3. 趋势变化观察表

**用途**：看行业和用户兴趣往哪里变化

**建议字段**：
| 字段 | 说明 |
|------|------|
| 趋势信号 | 观察到的变化信号 |
| 热词 | 相关热门关键词 |
| 用户评论 | 代表性用户评论 |
| 高频问题 | 用户反复问的问题 |
| 变化判断 | 这个信号是短期热点还是长期趋势？ |
| 对本品牌的影响 | 需要关注还是需要行动？ |

---

## 4. 三维对比矩阵

**用途**：把竞品/同行/趋势放在一起，形成结构化判断

**建议字段**：
| 维度 | 说明 |
|------|------|
| 交集 | 竞品、同行、趋势三者共同指向的方向 |
| 差异 | 竞品在做什么但同行没做 / 趋势在变但没人跟进 |
| 空白机会 | 尚未被充分占领的场景/卖点/表达方式 |
| 潜在风险 | 跟风哪些方向可能踩坑（同质化、不适合本品牌等） |
| 建议动作 | 基于以上判断的下一步 |

---

## 5. 7天/30天行动表

**用途**：把洞察转成可执行的内容测试计划

**7天快速测试表 — 建议字段**：
| 字段 | 说明 |
|------|------|
| 天数 | D1-D7 |
| 目标 | 当天的测试目标 |
| 具体动作 | 要做什么内容 |
| 内容形式 | 图文/视频/直播/笔记等 |
| 发布平台 | 小红书/抖音/公众号/天猫等 |
| 观察指标 | 用什么数据衡量效果 |
| 负责人 | 谁执行 |

**30天持续优化表 — 建议字段**：
| 字段 | 说明 |
|------|------|
| 周次 | W1-W4 |
| 阶段目标 | 本周目标 |
| 内容动作 | 具体做什么内容 |
| 测试变量 | 本周测试什么变量（标题/封面/场景等） |
| 观察指标 | 衡量指标 |
| 复盘时间 | 何时复盘 |
| 优化方向 | 根据复盘如何调整 |
