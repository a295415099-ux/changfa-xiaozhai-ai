# AI提示词模板（可直接复制使用）

本文档包含19个步骤对应的完整AI提示词。使用时将 `____` 替换为实际内容。

---

## 阶段一：定义范围

### 1. 明确对象 — 分析范围定义

```
请根据以下品牌信息，帮我明确本次内容洞察的分析范围：

品牌信息：____

请从以下维度输出分析范围：
- 品牌定位一句话总结
- 核心产品线
- 目标用户画像（年龄、性别、消费力、生活方式）
- 主要经营平台（小红书/抖音/公众号/天猫等）
- 建议观察的竞品类型（直接竞品/替代型竞品/内容竞品）
```

### 2. 拆解问题 — 三维问题框架

```
请把"内容营销洞察"拆成以下三类问题：

1. 竞品问题（竞品在讲什么、推什么、怎么讲）
   - 请给出3-5个关键子问题

2. 同行问题（同行在打什么场景、用什么表达方式）
   - 请给出3-5个关键子问题

3. 趋势问题（行业热点、用户偏好、表达方式往哪里走）
   - 请给出3-5个关键子问题

输出格式：每类子问题以列表呈现，每个子问题一句话说清。
```

### 3. 补充背景 — 品牌背景摘要

```
请将以下品牌背景整理成一页分析摘要，便于后续做竞品与趋势判断：

品牌信息：____

请输出：
- 品牌定位（一句话）
- 核心卖点（3-5个，按重要性排序）
- 目标用户画像（年龄、性别、消费力、生活方式、痛点）
- 价格带区间
- 主销平台与渠道特点
- 品牌调性（关键词3-5个）
```

### 4. 确认竞品 — 竞品识别与分层

```
请根据以下品牌信息，帮我识别三类竞品：

品牌信息：____

1. **直接竞品**：同类目、同价格带、同目标人群的品牌
2. **替代型竞品**：不同类目但满足同一需求的品牌
3. **内容竞品**：内容做得好的品牌，即使产品不直接竞争

每类列出3-5个品牌，并说明判断依据。
注意：不要只看销量竞品，也要看内容竞品。
```

### 5. 确认同行 — 同行品牌池

```
请列出和以下品牌同类目、同价格带、同平台打法的品牌：

品牌信息：____
类目：____
价格带：____
主要平台：____

请将同行品牌分为三类：
- **头部品牌**（行业前3-5）
- **腰部品牌**（稳定增长）
- **新锐品牌**（近一年快速增长）

每类列出3-5个品牌，附简要选择理由。
建议最终从中选定5-10个作为分析样本。
```

### 6. 确定趋势范围 — 趋势观察维度

```
请为以下行业设计一套趋势观察框架：

行业：____
目标用户：____
主要平台：____

请从四个维度定义观察范围：
1. **平台趋势**：热点话题、热门内容形式、算法变化
2. **内容趋势**：流行表达方式、视觉风格、内容结构变化
3. **消费趋势**：用户购买决策变化、新消费场景、价格敏感度变化
4. **情绪趋势**：用户主要情绪词、价值观表达变化

每个维度给出3-5个具体观察指标。
注意：趋势不要只写成一句"年轻化"。
```

## 阶段二：竞品分析

### 7. 抓取竞品内容 — 样本整理

```
请将以下竞品内容整理为结构化表格：

竞品内容：____

输出表格包含以下字段（每条内容为一行）：
| 发布时间 | 标题 | 平台 | 内容形式 | 主推产品 | 核心卖点 | 核心场景 |
| 情绪调性 | 正文摘要 | 互动数据（点赞/收藏/评论/转发） | 是否带货 |

如果没有完整数据，请标注为"未提供"。
建议每个品牌10-30条内容样本。
```

### 8. 竞品归类 — 内容分类表

```
请把以下竞品内容按多个维度进行归类：

竞品内容：____

请从以下6个维度分别归类，每类给出3-5个典型特征：
1. **主题**：内容的主题分类（如产品介绍、使用教程、品牌故事、用户证言等）
2. **产品**：主推产品的类型分布
3. **场景**：消费/使用场景分类
4. **情绪**：内容传递的情绪调性（如治愈、焦虑、惊喜、信任等）
5. **卖点**：核心卖点类型（如成分、功效、价格、设计、体验等）
6. **话术**：高频表达方式与句式

输出格式：每个维度一个分类表，包含类别名称、出现频次、典型特征、示例。
注意：先归类，再判断。不要跳过归类直接给结论。
```

### 9. 竞品策略分析 — 传播策略总结

```
请分析这些竞品内容的核心传播策略：

竞品样本池：____

请从以下5个维度输出分析结论：
1. **核心主题**：竞品反复在讲什么主题？高频重复的是什么？
2. **主推产品线**：竞品重点在推哪些产品？推产品的逻辑是什么？
3. **卖点表达**：竞品如何包装卖点？有什么共同套路？
4. **标题风格**：标题有什么规律？（如数字型、问答型、痛点型、结果型）
5. **转化逻辑**：从吸引点击→建立信任→引导下单的路径是什么？

每个维度给2-3个具体分析点和证据。
注意：关注"高频重复"的表达，那才是真正的策略。
```

### 10. 竞品话术提炼 — 高频话术库

```
请从以下文案中提炼竞品高频话术：

文案样本：____

请输出：
1. **高频关键词**（Top 20，按频次排序）
2. **高频句式**（5-8个典型句式，标注适用场景）
3. **情绪触发词**（哪些词在调动用户情绪？分正面/负面/中性）
4. **话术适用场景表**（什么场景下用什么话术）

注意：
- 不要只看词，还要看句式结构
- 不要照搬，重点学表达逻辑
```

## 阶段三：同行分析

### 11. 抓取同行内容 — 同行样本池

```
请整理以下同行品牌的内容：

同行内容：____

输出结构化表格，包含以下字段：
| 品牌 | 发布时间 | 平台 | 标题 | 封面主题 |
| 内容类型 | 核心场景 | 互动数据（点赞/收藏/评论） | 内容亮点 |

如果信息不完整，标注"需补充"。
注意：不仅看头部品牌，也看增长快的新锐品牌。
```

### 12. 同行场景分析 — 同行场景地图

```
请分析以下同行内容主要在打哪些消费场景：

同行内容样本：____

请从以下维度识别场景，并按出现频次从高到低排序：

1. **使用场景**：产品在什么情况下被使用？（如日常通勤、周末出游、居家放松）
2. **送礼场景**：什么节日、什么关系、什么预算？
3. **生活方式场景**：关联了哪种生活方式？（如精致生活、懒人经济、极简主义）
4. **情绪场景**：满足了什么情绪需求？（如焦虑缓解、自我奖励、社交归属）
5. **身份场景**：面向什么身份标签？（如新手妈妈、职场女性、运动爱好者）

每个场景补充2-3个典型表达示例。
注意：商家最容易忽略"场景表达"，这部分要特别详细。
```

### 13. 爆款因子分析 — 出圈内容规律

```
请分析以下高互动内容为什么更容易出圈：

高互动样本：____

请从以下5个维度输出结论，每个维度2-3个痛点：

1. **标题因子**：什么样的标题最容易引发点击？
2. **封面因子**：什么样的封面更吸引停留？
3. **场景因子**：什么场景更容易引发共鸣？
4. **情绪因子**：什么情绪最容易引发收藏/评论/转发？
5. **评论共鸣点**：评论区用户在讨论什么？有什么共同的痛点或需求？

最后，总结成3-5个可复用的爆款内容结构模板。

注意：
- 别只看播放量，收藏率和评论率比播放量更有价值
- 每个结论必须有样本证据支撑
```

## 阶段四：趋势分析

### 14. 趋势信号收集 — 趋势信号池

```
请结合以下资料，整理出该行业最近30-90天的趋势信号：

热词与热点：____
评论区高频词：____
用户讨论话题：____

请从以下维度输出趋势信号：
1. **热门话题**：用户在讨论什么新话题？
2. **关键词变化**：哪些关键词热度在上升？哪些在下降？
3. **用户新需求**：评论/问答中出现了什么新需求？
4. **生活方式变化**：用户的生活方式有什么新变化？
5. **表达方式变化**：用户的表达方式有什么新特点？（更直接？更情绪化？更专业？）

每个信号附1-2条原始样本作为证据。
注意：趋势要贴近用户，而不是只贴近行业报告。从评论区找线索。
```

### 15. 兴趣点变化分析 — 用户偏好转变

```
请根据以下内容与评论，判断用户兴趣点正在发生什么变化：

资料：____

请输出：
1. **兴趣迁移路径**：用户兴趣正在从什么转向什么？（如从"成分党"转向"体验派"）
2. **审美偏好变化**：视觉、表达风格有什么新偏好？
3. **决策因子变化**：用户做购买决策时更看重什么？（从价格→体验？从品牌→口碑？）
4. **证据支撑**：每个变化方向至少附2条原始评论或内容作为证据

注意：
- 一定要求AI"举证"，不能只给结论
- 要写出变化方向，不只是关键词堆积
```

## 阶段五：综合判断与行动

### 16. 三维对比 — 三维对比矩阵

```
请将以下三组分析结果做交叉对比：

竞品分析结果：____
同行分析结果：____
趋势分析结果：____

请输出：
1. **交集**：竞品、同行、趋势三者都在指向什么？
2. **差异**：哪些是竞品在做但同行没做的？哪些是趋势在变但同行还没跟上的？
3. **空白机会**：哪些场景/卖点/表达方式尚未被充分占领？
4. **潜在风险**：跟风哪些方向可能踩坑？

输出格式：四象限矩阵。

注意：这是从"看信息"到"做判断"的关键一步，不能跳过。
```

### 17. 提炼机会点 — 品牌机会点清单

```
请站在以下品牌立场，输出三类机会点：

品牌信息：____
三维对比结果：____

1. **可借鉴**（5-8条）
   - 哪些竞品/同行的打法可以直接学习？
   - 借鉴时需要注意什么？

2. **可差异化**（5-8条）
   - 哪些场景/卖点/表达方式是竞品覆盖不到或做得不好的？
   - 怎么做出自己的特色？

3. **不可照搬**（3-5条）
   - 哪些方向虽然流行，但不适合本品牌？
   - 为什么不适合？

注意：避免商家只会抄竞品，必须给差异化方向。
```

### 18. 转成策略 — 30天内容策略

```
请根据机会点清单，为以下品牌输出接下来30天的内容策略：

品牌信息：____
机会点清单：____

请输出：
1. **内容主题**：3-5个核心内容主题方向
2. **主推产品**：建议重点推哪些产品？推的逻辑是什么？
3. **核心场景**：建议主打哪2-3个消费场景？
4. **情绪价值**：内容应传递什么情绪调性？
5. **表达语系**：建议用什么语言风格？（如专业科普型、情感共鸣型、娱乐轻松型）

每个维度说明选择理由。
注意：先做方向，再写具体选题。不要一上来就写标题。
```

### 19. 生成行动计划 — 7天/30天行动表

```
请把以上洞察结果转成7天和30天的内容测试行动表：

所有分析结果：____

请输出两份行动表：

**7天快速测试表**
| 天数 | 目标 | 具体动作 | 内容形式 | 发布平台 | 观察指标 | 负责人 |

**30天持续优化表**
| 周次 | 阶段目标 | 内容动作 | 测试变量 | 观察指标 | 复盘时间 | 优化方向 |

总体要求：
- 前7天：快速测试2-3个方向，观察哪个方向数据更好
- 第2-4周：聚焦最优方向，放大产出
- 每周复盘一次，根据数据调整
- 备注：方便课程最后直接布置作业，颗粒度要够细
```
