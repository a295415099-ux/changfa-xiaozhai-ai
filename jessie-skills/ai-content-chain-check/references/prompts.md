# AI提示词模板

以下9个标准化提示词，执行各步骤时直接使用。替换 `{品牌名}`, `{核心SKU}`, `{内容样本}` 等占位符为实际数据。

## 1. 关键词种植检查

```
请根据以下内容样本，判断这条内容有没有清晰种下品牌词、品类词、卖点词、场景词，并说明用户看完后是否知道下一步搜什么。

品牌名：{品牌名}
核心SKU：{核心SKU}
内容标题：{内容标题}
内容正文/口播文案：{内容文案}
评论区关键信息：{评论区信息}
目标产品：{目标产品}

请输出：
1. 品牌词种植情况（是否出现、是否清晰、用户能否记住）
2. 品类词种植情况
3. 卖点词/功能词种植情况
4. 场景词种植情况
5. 综合判断：用户看完后是否知道搜什么？如果不知道，存在哪些关键词缺失？
6. 建议补充的关键词清单
```

## 2. 用户下一步动作推演

```
请站在目标用户视角，判断用户看完这条内容后最可能做什么：搜词、进主页、点商品、进直播间还是直接离开，并说明原因。

用户画像：{用户画像描述}
平台属性：{内容发布平台}
内容样本：{内容标题/链接}

请输出：
1. 用户看完后的第一反应/情绪
2. 最可能的下一步动作（单选）
3. 为什么是这个动作（列出1-3个理由）
4. 是否存在动作引导不足的问题
5. 理想的用户路径应该是什么
```

## 3. 搜索承接检查

```
请根据以下搜索导向词、商品标题、详情页、店铺页、活动页和直播间信息，判断用户搜到之后能不能被接住。

搜索导向词：{内容里种下的搜索词}
商品标题：{商品标题}
详情页核心卖点：{详情页关键信息}
店铺页表达：{店铺页关键信息}
直播间标题/主题：{直播标题}
活动页信息：{活动页信息}

请输出：
1. 搜索词与商品标题的匹配度（是否包含、是否靠前）
2. 详情页是否回应了搜索意图
3. 店铺页整体表达是否与搜索预期一致
4. 直播/活动页是否承接了搜索流量
5. 承接断点识别：哪个环节最可能造成用户流失
```

## 4. 表达一致性检查

```
请对比内容里的卖点、场景、关键词，和商品页/店铺页/活动页/直播间里的表达是否一致，并指出哪里断裂。

内容文案：{内容文案}
商品页标题/详情：{商品页信息}
店铺页表达：{店铺页信息}
活动页文案：{活动页信息}
直播口播：{直播口播信息}

请输出：
1. 内容表达 vs 页面表达 逐项对比表
2. 一致性评分（1-5分）
3. 断裂点标注（具体哪里不一致）
4. 不一致可能导致的用户困惑
5. 修改建议：让页面表达与内容保持"一套语言"
```

## 5. 利益点检查

```
请检查以下承接页是否讲清楚为什么买、现在为什么买，并指出用户进来后最容易流失的原因。

商品页信息：{商品页链接/截图描述}
活动页信息：{活动页链接/描述}
利益点机制：{优惠、赠品、限时等}
直播话术：{直播话术关键段落}

请输出：
1. "为什么买"是否清晰（产品价值表达）
2. "现在为什么买"是否清晰（紧迫感/优惠/限时）
3. 用户最容易流失的3个原因
4. 利益点优化建议
5. 是否需要补充信任证据（评价、背书、销量等）
```

## 6. 转化动作检查

```
请检查以下页面或直播间是否有清晰的转化动作指引，例如加购、领券、咨询、进直播间等，并判断是否足够明确。

页面结构：{页面关键模块描述}
CTA文案：{CTA按钮/文案}
直播间动作设计：{直播间引导动作}

请输出：
1. 当前已有的转化动作清单
2. 每个CTA的清晰度评分
3. 是否存在"不知道下一步该做什么"的问题
4. CTA改进建议（位置、文案、数量）
5. 建议补充的转化动作
```

## 7. 节奏同步检查

```
请根据以下时间信息，判断内容发布时间、搜索优化、直播排期、活动页上线是否同步，并指出时间错位问题。

内容发布时间：{发布时间}
搜索词优化时间：{搜索优化时间}
直播间排期：{直播排期}
活动页上线时间：{活动页上线时间}
活动节奏：{活动节奏说明}

请输出：
1. 各时间节点的对齐情况
2. 是否存在"内容先发，承接后补"的问题
3. 时间错位导致的流量浪费预估
4. 理想的时间节奏同步方案
5. 后续内容发布的时间节奏建议
```

## 8. 断链类型归类

```
请根据以上检查结果，帮我归类这条链路目前属于哪几种断链：关键词断链、搜索承接断链、利益点断链、节奏错位断链，并给出优先级。

前面所有检查结果：{汇总前面的分析结果}

请输出：
1. 已识别的断链类型及严重程度
2. 各断链类型的优先级排序
3. 每种断链的根本原因
4. 如果只修一个问题，应该先修哪个？为什么？
```

## 9. 断链修复建议

```
请基于以上断链问题，为我输出修复建议，包括内容怎么补词、搜索怎么补、页面怎么补、直播怎么接、节奏怎么调。

前面所有分析结果：{汇总前面的分析结果}

请输出一份结构化修复方案：
1. 内容侧修复：哪些内容需要补什么关键词
2. 搜索侧修复：搜索词优化建议
3. 页面侧修复：商品页/店铺页/活动页需要改什么
4. 直播侧修复：直播话术/排期调整建议
5. 节奏调整：时间线重新对齐建议
6. 优先级排序：按影响程度和实现难度排序
7. 预期效果预估
```
