# AI三类关键词统一SOP

## 前置规则

执行前必须先读以下 references 文件：

- `assets/keyword-unified-template.xlsx` — 三类关键词统一填写模板（可直接分发给用户填写）

## 核心认知

这不是堆词，而是统一品牌表达系统。

三类关键词各有独立作用，不能混用：

| 类型 | 解决什么 | 本质问题 |
|------|---------|---------|
| 搜索词 | 被找到 | 用户会怎么找你 |
| 卖点词 | 被说服 | 产品凭什么值得买 |
| 场景词 | 被触发 | 什么情境下最容易心动 |

最终目标：让用户看到的内容、搜到的结果、进入的页面形成一体化感知。

## 执行流程概览（19步）

```
阶段0-2：认知对齐
  ├─ 0. 明确本次任务（统一品牌表达系统，不是堆词）
  ├─ 1. 输入品牌基础信息
  └─ 2. 明确关键词三分类

阶段3-7：现状盘点
  ├─ 3. 梳理现有表达资产
  ├─ 4. 提取现有搜索词
  ├─ 5. 提取现有卖点词
  ├─ 6. 提取现有场景词
  └─ 7. 识别表达断裂点

阶段8-11：重构优化
  ├─ 8. 从用户视角重构搜索词
  ├─ 9. 从成交视角重构卖点词
  ├─ 10. 从种草视角重构场景词
  └─ 11. 建立三类词映射关系

阶段12-16：统一与应用
  ├─ 12. 统一主表达口径
  ├─ 13. 输出内容侧应用
  ├─ 14. 输出搜索侧应用
  ├─ 15. 输出成交侧应用
  └─ 16. 形成关键词总表

阶段17-18：验证与迭代
  ├─ 17. 输出首轮测试计划
  └─ 18. 建立复盘机制
```

详细步骤见 `references/sop-main-table.md`

## 执行指南

### 开始前必问

在开始执行SOP前，必须收集以下信息（根据用户提供情况选择是否提问）：

1. **品牌基础信息**：品牌名、品牌定位、产品类型、核心卖点、目标用户、主要平台、价格带
2. **现有素材**：是否有现有内容（小红书/抖音/天猫/京东/私域）、商品标题、详情页、客服话术可参考
3. **任务范围**：是完整走一遍全部19步，还是聚焦在某个特定阶段

### 访谈规则

如果用户没有提供足够的品牌信息，使用以下问题收集（最多5个问题，只问一轮）：

**Q1. 品牌基础信息**
请提供：品牌名、产品类型、核心卖点、目标用户画像、主要销售平台、价格带

**Q2. 现有素材情况**
是否有以下素材可以分析？
A) 有——已整理好的品牌介绍、产品资料
B) 有——但分散在小红书/抖音/天猫等多平台
C) 几乎没有——从零开始梳理

**Q3. 当前痛点**
目前最想优先解决的问题是？
A) 内容有播放但没搜索
B) 有搜索但页面接不住
C) 内容和页面的语言不统一
D) 不确定用户到底在搜什么词

**Q4. 任务范围**
希望执行的深度是：
A) 完整走一遍全部19步
B) 聚焦现状盘点（阶段3-7）
C) 聚焦重构优化（阶段8-11）
D) 聚焦应用落地（阶段12-16）

**Q5. 输出偏好**
期望的输出形式：
A) 仅文本表格输出
B) 需要填写到Excel模板中
C) 两者都需要

### 执行原则

1. **严格按阶段顺序执行**，不跳过阶段
2. **每完成一个阶段，向用户展示该阶段的输出结果**，确认后再进入下一阶段
3. **用户说"可以了"或"下一个"时，才进入下一阶段**——不要一口气跑完
4. **使用 references/prompt-templates.md 中的提示词模板**向AI提问，保持输出结构一致
5. **最终交付**必须包含 `references/delivery-checklist.md` 中列出的全部9项成果

### 输出格式要求

每阶段输出包含：
- **本阶段目标**：一句话回顾
- **核心发现/输出**：结构化表格
- **下一步**：预告下一阶段做什么
- **确认问题**：问用户是否继续或需要调整

### 使用模板

如果用户需要填写Excel模板，使用 `assets/keyword-unified-template.xlsx`：
- 直接复制该模板文件到工作目录
- 使用 openpyxl 填充数据
- 完成后用 message 工具发送给用户
