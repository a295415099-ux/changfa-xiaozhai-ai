# 三类关键词定义

## 概述

AI三类关键词统一SOP的核心认知：搜索词、卖点词、场景词各有独立作用，不能混用。

---

## 三类关键词标准理解

### 1. 搜索词

- **核心作用**：解决被找到
- **本质回答的问题**：用户会怎么找你
- **常见表现位置**：搜索框、商品标题、标签、问答、评论
- **特点**：
  - 必须站在用户视角
  - 含类目词、需求词、场景词、问题词、送礼词
  - 是用户认知的语言，不是品牌内部语言

### 2. 卖点词

- **核心作用**：解决被说服
- **本质回答的问题**：你的产品凭什么值得买
- **常见表现位置**：内容标题、正文、详情页、直播口播
- **特点**：
  - 必须从参数翻译成用户利益
  - 区分功能描述、用户收益、差异化价值
  - 让用户听得懂、记得住、愿意信

### 3. 场景词

- **核心作用**：解决被触发
- **本质回答的问题**：用户在什么情境下最容易心动
- **常见表现位置**：种草内容、短视频、图文封面、生活方式表达
- **特点**：
  - 不只是地点，更是情绪与动机
  - 包含日常使用、送礼、出行、办公、家庭、情绪陪伴等
  - 体现地点+情绪+动机+节点

---

## 核心问题

这三类词回答了品牌表达系统最关键的三个问题：

1. **用户怎么找你？** → 搜索词
2. **你凭什么值得买？** → 卖点词
3. **什么情境下最容易被打动？** → 场景词

三者不是孤立的词汇表，而是为解决问题而设计的一张能力表。最终目标是形成统一的品牌对外表达系统。
