# 核心概念速查

## 一句话总结

**任何一个环节断掉，内容流量都会变成"看过就走"。**

## 核心诊断逻辑

很多品牌不是没有做内容，也不是没有投流量，而是内容、搜索、承接、投放、复购、团队之间没有形成一条完整链路。

## 完整链路

```
内容种草 → 搜索回流 → 店铺承接 → 投放放大 → 复购沉淀 → 团队协同
```

## 六大断点速查

| 断点 | 核心问题 | 判断标准 |
|------|----------|----------|
| 种草弱 | 内容是否能真正种草？ | 用户看完是否产生"这说的不就是我吗"的感觉 |
| 搜索弱 | 种草和搜索是否连上？ | 外部关键词与淘内搜索词是否统一 |
| 承接弱 | 流量进店能不能接住？ | 详情页/主图/评价是否延续了种草理由 |
| 投放弱 | 知不知道什么内容值得放大？ | 有无优质素材筛选标准、投放与内容是否共用指标 |
| 复购弱 | 首单后有没有后续经营？ | 有无会员/私域/复购内容机制 |
| 协同弱 | 团队是否围绕同一目标？ | 关键词/目标/指标/行动/交付是否统一 |

## 适用对象

品牌老板、品牌负责人、内容负责人、电商负责人、投放负责人、客服负责人、培训讲师。
