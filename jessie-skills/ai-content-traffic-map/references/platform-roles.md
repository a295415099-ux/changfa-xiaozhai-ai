# 全域全平台内容流量地图：平台角色拆解

## 五大平台类型

| 平台类型 | 代表平台 | 主要任务 | 适合内容 | 典型承接动作 |
|----------|---------|----------|----------|-------------|
| 种草平台 | 小红书 / 抖音内容层 | 激发兴趣、放大场景、建立初始好感 | 场景种草、体验分享、痛点唤醒、用户故事 | 记住关键词、收藏、评论、去搜索 |
| 信任平台 | 视频号 / 公众号 / 品牌号 | 建立信任、拉近关系、解释价值 | 品牌故事、深度讲解、专家背书、使用方法 | 点击主页、咨询、进直播、进私域 |
| 搜索承接平台 | 淘宝 / 天猫 / 京东 / 抖音商城搜索 | 承接关键词搜索与商品决策 | 商品标题、详情页、问大家、评价内容 | 进商品页、加购、对比、咨询 |
| 成交平台 | 直播间 / 活动页 / 店铺首页 | 放大转化、处理犹豫、促成下单 | 直播讲解、限时机制、套餐逻辑、FAQ | 加购、领券、下单 |
| 私域平台 | 企业微信 / 社群 / 小程序 | 持续跟进、反复触达、沉淀用户 | 答疑、提醒、福利、复购引导 | 咨询、复购、社群裂变 |

---

## 各平台角色详解

### 1. 种草平台（小红书 / 抖音内容层）

**核心任务**：激发兴趣、放大场景、建立初始好感

**内容特征**：
- 以场景化为核心——用户在什么情境下需要这个产品
- 重体验感受而非参数罗列
- 强调"痛点唤醒"和"使用故事"
- 视觉表达优先

**典型承接动作**：
- 让用户记住关键词
- 引导收藏、评论互动
- 引导"去搜索"——这是种草平台最关键的承接动作
- 不要直接在种草内容里做成交

**常见问题**：
- 把种草内容做成了硬广
- 种草完没有引导搜索动作
- 关键词不统一，种草词和搜索词对不上

---

### 2. 信任平台（视频号 / 公众号 / 品牌号）

**核心任务**：建立信任、拉近关系、解释价值

**内容特征**：
- 深度内容为主——讲透"为什么选我们"
- 品牌故事、创始人故事
- 专家背书、第三方评测
- 使用教程、方法分享
- 与私域联动——信任平台的用户更容易进入私域

**典型承接动作**：
- 引导点击主页了解更多
- 引导咨询（客服/企微）
- 引导进入直播间
- 引导进入私域/社群

**常见问题**：
- 信任内容和种草内容混淆——信任内容应该更深入
- 没有在信任内容中设置明确的下一步动作

---

### 3. 搜索承接平台（淘宝 / 天猫 / 京东 / 抖音商城搜索）

**核心任务**：承接关键词搜索与商品决策

**内容特征**：
- 商品标题必须包含用户会搜索的关键词
- 详情页需要回应种草内容中设置的期待
- 评价内容和"问大家"是用户决策的关键内容
- 主图/视频需要与种草内容的调性保持一致

**典型承接动作**：
- 进商品页
- 加购
- 对比同店其他商品
- 咨询客服

**常见问题**：
- 商品标题关键词和种草引导的关键词不匹配
- 详情页没有接住种草建立的期待
- 评价内容缺乏运营（差评没有回应）

---

### 4. 成交平台（直播间 / 活动页 / 店铺首页）

**核心任务**：放大转化、处理犹豫、促成下单

**内容特征**：
- 直播脚本需要包含种草阶段提到的卖点和场景
- 直播间需要能处理"我在小红书/抖音看到过"这些信息
- 活动页需要营造紧迫感
- FAQ需要覆盖从种草到购买的常见疑虑

**典型承接动作**：
- 加购
- 领券
- 下单
- 引导复购/连带购买

**常见问题**：
- 直播间讲的和种草讲的不是同一套逻辑
- 活动页的卖点排序和种草阶段不一致
- 没有处理用户的犹豫和不信任

---

### 5. 私域平台（企业微信 / 社群 / 小程序）

**核心任务**：持续跟进、反复触达、沉淀用户

**内容特征**：
- 答疑内容——解决购买后的使用问题
- 提醒内容——复购周期、活动通知
- 福利内容——专属优惠、社群活动
- 复购引导——基于使用周期的推荐

**典型承接动作**：
- 咨询
- 复购
- 社群裂变
- 老带新

**常见问题**：
- 私域内容和公域内容各说各话
- 私域没有承接好公域建立的预期
- 私域只做推送不做服务和关系维护

---

## 平台协同核心原则

1. **不是一套内容到处平移**：不同平台承担不同任务，内容形式、表达方式都应适配平台特性
2. **搜索词要统一**：种草平台引导的关键词，必须能在搜索承接平台搜到
3. **承接要前置设计**：不是发完内容再想承接，而是在做内容之前就想好承接
4. **信任是分层建立的**：种草建好感 → 信任建认知 → 搜索建决策 → 成交建行动 → 私域建关系
5. **平台间要有"接力"关系**：用户从种草平台到搜索平台到成交平台，体验应该是一体的
