# 短视频脚本 Check List — 一键检查清单

生成脚本后逐项核对，每条通过方为合格。

## 1. 前3秒检查
- [ ] 标题让目标人群一眼识别「和我有关」
- [ ] 开头钩子（前3秒）能在刷视频时让人停下
- [ ] 首帧画面有视觉冲击力

## 2. 画面可拍性检查
- [ ] 每个分镜有具体画面描述（非抽象概念）
- [ ] 有明确动作描述
- [ ] 拍摄提示含角度/灯光/时长建议
- [ ] 团队凭此表能直接拍

## 3. 卖点可理解性检查
- [ ] 口播卖点是用户语言，非品牌参数堆砌
- [ ] 非专业人士能一次听懂
- [ ] 每个卖点有对应使用场景

## 4. 关键词与承接检查
- [ ] 核心关键词在口播+字幕中充分露出
- [ ] 结尾行动引导指向明确搜索词或页面
- [ ] 承接路径清楚（商品页/店铺/直播）
