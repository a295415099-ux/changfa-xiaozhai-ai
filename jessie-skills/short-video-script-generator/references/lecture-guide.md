# 短视频脚本生成 Skill 课程讲稿参考

## 上屏核心文案

短视频不是把卖点念一遍，而是先解决：**前3秒停留、中段理解、后段信任、结尾行动。**

## 5个核心结构

1. **标题** — 让用户一眼判断「这条内容和我有关吗」
2. **开头钩子** — 前3秒抓停留
3. **场景画面** — 具体可拍，非抽象概念
4. **卖点表达** — 从品牌语言转成用户语言
5. **行动引导** — 自然导向下一步动作

## Prompt输出要求

一次完整生成须包含：
- 视频标题（≥5个备选）
- 前3秒开头钩子（≥3个备选）
- 3-5个分镜画面
- 每个分镜口播文案
- 每个分镜字幕文字
- 对应卖点说明
- 结尾行动引导
- 淘内承接建议

## 核心输出物

《短视频脚本表》

## 现场动作要求

各组围绕一个核心产品，现场完成一条**可拍、可讲、可承接**的短视频脚本。
