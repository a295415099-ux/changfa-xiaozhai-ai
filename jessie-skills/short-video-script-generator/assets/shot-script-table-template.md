# 短视频脚本表模板

用于阶段17输出《短视频脚本表》初版。每次生成一条完整的短视频脚本时，严格按此表格式输出。

## 脚本表通用格式

| 镜头 | 画面 | 口播 | 字幕 | 对应卖点 | 拍摄提示 |
|------|------|------|------|----------|----------|
| 镜头1 | （具体画面描述：场景、人物动作、产品状态） | （口播文案） | （屏幕显示文字） | （该镜头传达的核心卖点） | （拍摄角度、灯光、时长等） |
| 镜头2 | ... | ... | ... | ... | ... |
| 镜头3 | ... | ... | ... | ... | ... |
| 镜头4 | ... | ... | ... | ... | ... |
| 镜头5 | ... | ... | ... | ... | ... |

## 各字段填写要求

### 画面
- 必须具体、可执行，不是抽象概念
- 描述：场景环境、人物/产品位置关系、关键动作、产品特写/展示方式
- 错误示例：「展示产品优势」
- 正确示例：「俯拍厨房台面，一只手拿起产品挤出黄豆粒大小于掌心，另一只手蘸取涂抹于手背，镜头推进拍涂抹前后对比」

### 口播
- 完整的语音文案，可以直接照着念
- 语言风格：口语化、有节奏感、像在跟朋友聊天
- 每段口播控制在10-25秒之间

### 字幕
- 精简版口播 + 强调关键词
- 每屏不超过2行
- 关键卖点处可加粗/变色提示

### 对应卖点
- 该镜头要传达的1个核心卖点
- 避免一个镜头塞多个卖点

### 拍摄提示
- 含以下要素：景别（特写/近景/中景/全景）、角度（平视/俯拍/仰拍）、灯光建议、预估时长、注意事项

| 镜头3 | 近景，面部特写，展示使用后水润效果 | 「保湿一整天，到下午脸还是软软的。」 | 保湿一整天，脸还是软软的 | 效果卖点 | 侧光展示皮肤光泽，3秒 |
