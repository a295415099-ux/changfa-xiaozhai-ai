# 品牌基础信息回收清单

使用本技能前，从用户处收集以下信息。如某项暂无，标注"待补充"继续推进。

## 必填项

### 1. 品牌基础
- [ ] 品牌名称
- [ ] 品牌定位（一句话）
- [ ] 品牌slogan

### 2. 目标人群
- [ ] 主要目标用户画像
- [ ] 成交用户特征
- [ ] 典型用户使用场景

### 3. 商品/产品
- [ ] 核心商品列表
- [ ] 主要卖点
- [ ] 商品链接（天猫/京东/小程序等）

### 4. 现有内容
- [ ] 品牌介绍文档
- [ ] 产品详情页
- [ ] 小红书/抖音/视频号/B站/知乎/公众号账号
- [ ] 历史种草内容
- [ ] 达人合作内容

### 5. 用户反馈
- [ ] 评论区高频问题
- [ ] 客服问答记录
- [ ] 用户好评/差评摘录

## 选填项

### 6. 竞品信息
- [ ] 主要竞品名称
- [ ] 用户常对比的竞品问题

### 7. 团队资源
- [ ] 内容团队规模和分工
- [ ] 可投入内容制作的资源

## 资料不足处理

如果用户提供的资料严重不足（如缺少5项以上必填项），在阶段1结束后暂停，建议用户补充后再继续。
