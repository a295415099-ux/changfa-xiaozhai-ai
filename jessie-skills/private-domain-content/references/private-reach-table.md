# 私域触达内容表模板

## 使用说明

此表是 Skill 执行步骤 11 的核心输出模板。围绕触达对象、节点、状态、内容、方式和转化动作六个维度完整设计私域触达方案。

| 触达对象 | 触达节点 | 用户状态 | 内容主题 | 触达方式 | 转化动作 |
|----------|----------|----------|----------|----------|----------|
| 新客 | 下单后 1-3 天 | 刚购买 | | | |
| 新客 | 收货后 3-7 天 | 开始使用 | | | |

## 各列填写指引

### 触达对象
- 新客：首次购买，无复购记录
- 老客：有复购记录，活跃中
- 沉默用户：超过 30 天未互动/未购买
- 会员用户：已加入会员体系

### 触达节点
基于用户旅程的关键时间点，如：
- 下单后 1-3 天、收货后 3-7 天
- 使用周期 1/3 处、2/3 处
- 复购周期前 7 天、前 3 天
- 大促前 7 天、前 1 天
- 沉默 30 天、60 天、90 天

### 用户状态
- 刚购买 → 需要教育引导使用
- 开始使用 → 需要维护关系
- 可能需要补充 → 需要推荐搭配品
- 即将复购 → 需要复购提醒
- 可能流失 → 需要召回

### 内容主题
从 6 类私域内容中选择匹配的：
- 使用提醒 / 场景复购 / 搭配推荐 / 用户反馈 / 活动召回 / 知识教育

### 触达方式
- 短信 / 会员消息 / 微信/企微 / 客服触达 / 私域群 / 直播提醒

### 转化动作
- 正确使用 / 评价沉淀 / 加购关联品 / 复购下单 / 回店 / 进直播间

## 扩展行建议

根据实际业务复杂度，可在以上 5 行基础上扩展：
- 新客增加「使用后 14 天」节点
- 老客增加「会员日/品牌日」节点
- 沉默用户增加「60 天/90 天」节点
- 会员用户增加「会员权益到期前」节点
