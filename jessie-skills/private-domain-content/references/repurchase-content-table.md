## 各列填写指引

### 复购节点
- 首购后：用户刚完成首次购买，需要引导正确使用
- 使用中：用户正在使用产品，开始体验效果
- 用完前：产品即将用完，需要提醒补货
- 大促前：大促活动前，用户有囤货心理
- 沉默期：用户长期未购买，需要重新激活

### 用户状态
- 不知道怎么用 → 需要教育 + 引导
- 开始体验 → 需要维护 + 收集反馈
- 可能需要补货 → 需要提醒 + 推荐
- 有囤货需求 → 需要活动信息 + 组合推荐
- 长期未买 → 需要召回 + 重新建立连接

### 内容主题
从 6 类私域内容中选择：
- 使用提醒 / 场景复购 / 搭配推荐 / 用户反馈 / 活动召回 / 知识教育

### 推荐产品
- 同款复购（消耗品）
- 搭配产品（关联推荐）
- 升级款（更高客单）
- 新品（尝鲜推荐）
- 套装/组合（提高客单）

### 转化动作
- 正确使用 → 引导到使用教程/详情页
- 评价沉淀 → 引导写评价/晒单
- 加购关联品 → 加入购物车
- 复购下单 → 直接购买
- 回店 → 进店浏览
- 进直播间 → 观看直播
