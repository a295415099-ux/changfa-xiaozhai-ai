# SOP主表详细步骤

## 阶段0：明确本次任务

- **任务目标**：统一认知：60天陪跑不是附加福利，而是结果落地的后半段
- **具体动作**：明确目标：把接下来60天拆成清晰阶段任务，让团队知道怎么按节奏推进
- **输入内容示例**：品牌名、课程主题、训练营目标、618/双11等大促节奏
- **输出结果**：60天训练营任务定义
- **注意事项**：开场要强调"课堂是起点，60天才是结果真正长出来的地方"

---

## 阶段1：回收线下课核心成果

- **任务目标**：先把线下课上已经做出来的成果拉回来
- **具体动作**：汇总课程地图、优先级表、断链识别、承接优化、7天行动清单、经营判断表等成果
- **输入内容示例**：课堂表单、作业、分工表、判断表
- **输出结果**：线下课成果汇总表
- **注意事项**：60天不是重新开始，而是承接线下课结果继续往前跑

### 建议回收的课堂成果清单
1. 课程地图（内容/搜索/承接全景图）
2. 内容优先级表
3. 断链识别表（内容断链检查）
4. 承接优化动作表（店铺承接）
5. 7天行动清单（回店SOP）
6. 经营判断表（蓄水判断/趋势判断等）
7. 三角色分工表
8. 关键词统一表
9. 外种内收链路图
10. 内容协同流程表

---

## 阶段2：确认60天总目标

- **任务目标**：明确这60天到底要跑出什么结果
- **具体动作**：让AI帮助判断：这60天最核心的经营目标、阶段目标和观察指标是什么
- **输入内容示例**：品牌目标、SKU目标、618/大促节奏、内容目标、店铺目标
- **输出结果**：60天总目标表
- **注意事项**：没有总目标，任务就会变成碎片化推进

### 总目标表结构
| 维度 | 总目标 | 阶段里程碑 | 观察指标 |
|------|--------|-----------|---------|
| 经营目标 | | | |
| 内容目标 | | | |
| 搜索目标 | | | |
| 店铺目标 | | | |
| 团队目标 | | | |

---

## 阶段3：拆周期1 —— 统一与诊断

- **任务目标**：先把"理解"和"表达"真正接入业务
- **具体动作**：围绕统一表达、统一词系统、第一轮卡点校正，明确本周期任务
- **输入内容示例**：词表、表达模板、品牌卡点、断链识别表
- **输出结果**：周期1任务表
- **注意事项**：周期1的核心不是大量做内容，而是把底层系统先对齐

### 周期1任务表示例
- 统一品牌表达口径（搜索词、卖点词、场景词）
- 完成关键词统一映射
- 识别并记录当前内容表达断裂点
- 建立统一词模板供团队使用
- 检查店铺承接基础（标题首屏、商品页）

---

## 阶段4：拆周期2 —— 内容与蓄水

- **任务目标**：让内容真正开始服务大促前蓄水
- **具体动作**：明确高优先级内容推进、搜索回流观察、第一轮内容效果跟踪
- **输入内容示例**：内容优先级表、内容包装、搜索回流观察表
- **输出结果**：周期2任务表
- **注意事项**：周期2的核心是内容开始真正进入经营节奏

### 周期2任务表示例
- 推进Top5高优先级内容方向
- 发布蓄水内容（按内容优先级表推进）
- 开始观察搜索回流数据
- 跟踪第一轮内容效果（种草力、互动量）
- 内容包装优化（标题、利益点、CTA）

---

## 阶段5：拆周期3 —— 外种内收与承接优化

- **任务目标**：把站外种草和站内承接真正接起来
- **具体动作**：检查回淘链路、优化商品页与店铺页承接、加强内容与站内转化连接
- **输入内容示例**：外种内收链路图、店铺承接优化动作表、内容到承接检查表
- **输出结果**：周期3任务表
- **注意事项**：这一阶段是从"内容在跑"进入"内容开始接结果"

### 周期3任务表示例
- 逐项检查回淘路径是否通畅
- 优化商品详情页卖点承接
- 调整店铺首页/活动页内容一致性
- 加强内容评论区引导（搜索词+店铺入口）
- 检查搜索→承接页→下一步动作链路

---

## 阶段6：拆周期4 —— 复盘与调整

- **任务目标**：让真实经营反馈进入下一轮优化
- **具体动作**：明确看哪些动作有效、哪些内容值得加码、哪些环节需要修正
- **输入内容示例**：经营判断表、内容表现、店铺反馈、承接数据
- **输出结果**：周期4任务表
- **注意事项**：周期4的核心是从"执行"进入"策略调整"

### 周期4任务表示例
- 复盘大促/618期间内容资产表现
- 识别高转化内容、高互动内容、可投放素材
- 沉淀有效关键词到品牌词库
- 输出下半年内容放大计划
- 修正效果不佳的动作方向

---
