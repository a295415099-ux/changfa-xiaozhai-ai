# 老板版摘要模板

| 摘要项 | 内容 |
|--------|------|
| **一句话结论** | 链路画出来不算，真正接住才算。 |
| **当前最明显的承接问题** | [填写：1-3个最关键的断点] |
| **最先该补的那一段** | [填写：P0优先修复项] |
| **为什么这个问题最影响转化** | [填写：对成交的直接影响分析] |
| **今晚最值得先推进的修复动作** | [填写：今晚可执行的具体动作] |
| **下周要验证的关键指标** | [填写：搜索回流量/商品页访问量/加购率等] |

## 使用说明

此模板用于在完成内容到承接检查后，向决策者（老板/负责人）快速汇报核心发现。

要点：
- 简洁，一页讲清楚
- 聚焦"最关键的问题"和"最先该做什么"
- 不要罗列所有细节，只给最重要的判断和动作
