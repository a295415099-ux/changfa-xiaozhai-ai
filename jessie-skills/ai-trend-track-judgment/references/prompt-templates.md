# AI趋势赛道判断 — 提示词模板

以下提示词可直接复制使用，每次使用应根据品牌实际情况补充对应信息。

---

## 1. 用户趋势判断

```
请根据以下评论区、问大家、客服问题和用户反馈，帮我总结这个赛道的用户最近更关注什么、
表达方式有没有变化、决策逻辑有没有变化。

[粘贴评论区内容]
[粘贴问大家内容]
[粘贴客服问题记录]
[粘贴消费反馈数据]
```

**预期输出**：用户趋势判断表

---

## 2. 用户关注点变化

```
请判断这个赛道用户的关注点最近从"关注什么"转向"更关注什么"，并说明变化背后的原因。

[粘贴用户高频问题]
[粘贴咨询记录]
[粘贴搜索行为数据]
```

**预期输出**：用户关注点变化表

---

## 3. 内容趋势判断

```
请根据以下平台热点、高表现内容和同行爆款内容，判断近期哪些内容形式、内容风格、
表达方式更容易被平台和用户接受。

[粘贴平台热点]
[粘贴高表现内容样本]
[粘贴同行爆款内容]
```

**预期输出**：内容趋势判断表

---

## 4. 内容形式变化

```
请判断图文、短视频、直播切片、对比测评、口播种草、场景共鸣等内容形式在当前赛道里的效率变化，
并说明哪种更适合当前阶段。

[粘贴内容样本]
[粘贴分发特征数据]
[说明赛道属性]
```

**预期输出**：内容形式变化表

---

## 5. 赛道趋势判断

```
请根据行业信息、竞品布局、用户趋势和内容趋势，判断这个赛道里哪些细分方向在升温，
哪些方向已经开始红海，哪些方向适合提前卡位。

[粘贴行业信息]
[粘贴竞品布局]
[粘贴用户趋势结果]
[粘贴内容趋势结果]
```

**预期输出**：赛道趋势判断表

---

## 6. 值得追方向

```
请从以上趋势结果中，找出至少1个当前值得追的趋势方向，并说明为什么值得跟进。

[粘贴用户趋势结果]
[粘贴内容趋势结果]
[粘贴赛道趋势结果]
[说明品牌能力]
```

**预期输出**：值得追方向表

---

## 7. 谨慎跟方向

```
请从以上趋势结果中，找出至少1个表面热闹但需要谨慎跟的方向，并说明为什么不建议盲目跟。

[粘贴热点内容]
[粘贴红海方向]
[说明品牌能力]
```

**预期输出**：谨慎跟方向表

---

## 8. 提前卡位方向

```
请从以上趋势结果中，找出至少1个适合品牌提前卡位的方向，并说明为什么现在布局更有价值。

[粘贴赛道变化信息]
[粘贴竞品空档信息]
[说明品牌资源]
```

**预期输出**：提前卡位方向表

---

## 9. 品牌与节点匹配度判断

```
请判断以下趋势方向与品牌能力、品牌调性、供应链、内容能力以及618节点是否匹配，并说明原因。

[粘贴品牌定位]
[说明团队资源]
[说明节点目标]
[粘贴趋势方向列表]
```

**预期输出**：品牌匹配度判断表 + 节点匹配度判断表

---

## 10. 首轮动作建议

```
请基于以上趋势判断结果，为我输出首轮动作建议：哪些方向先观察、哪些先测试、哪些先卡位、
哪些可以加大投入。

[粘贴趋势结论]
[说明品牌资源]
[说明时间节点]
```

**预期输出**：首轮动作建议表
