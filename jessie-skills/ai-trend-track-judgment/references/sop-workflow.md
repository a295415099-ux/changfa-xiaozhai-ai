# AI趋势赛道判断表 — SOP全流程

## 概述

此SOP帮助品牌从"短期内容动作"升级到"赛道方向判断"，看清哪些方向正在上升、哪些方向已经拥挤、哪些方向值得提前卡位。

核心目标：解决"只会追热点、不会做赛道判断""看见热闹但分不清值不值得跟""不知道哪些方向适合品牌提前布局"等问题。

核心输出物：趋势赛道判断任务定义 / 赛道定义表 / 用户趋势判断表 / 内容趋势判断表 / 赛道趋势判断表 / 值得追方向表 / 谨慎跟方向表 / 提前卡位方向表 / 趋势赛道判断表 / 首轮动作建议表。

---

## 18步完整SOP流程

### 阶段0：明确本次任务
**目标**：统一认知——这一步不是看单条内容，而是看赛道方向。明确目标：判断赛道里哪些方向在上升、哪些方向已拥挤、哪些方向值得提前卡位。
**输入**：品牌名、所属赛道、核心SKU、课程主题
**输出**：趋势赛道判断任务定义
**注意**：这张表更偏中观判断，不是短期选题表。

### 阶段1：明确判断对象
**目标**：先界定这次看的是哪个赛道、哪个细分方向。
**具体动作**：明确所属行业、类目、细分赛道、典型竞品与典型用户。
**输入**：品牌品类、行业名称、细分类目、核心竞品
**输出**：赛道定义表
**注意**：不先界定赛道，趋势判断很容易泛化。

### 阶段2：回收前面洞察结果
**目标**：把前面的信息带入趋势判断。
**具体动作**：汇总前面三维洞察、内容机会、关键词、用户反馈结果。
**输入**：竞品分析表、用户需求表、内容机会表
**输出**：前序信息汇总表
**注意**：趋势判断不是凭空做，要建立在已有洞察上。

### 阶段3：判断用户趋势
**目标**：看用户最近在变什么。
**具体动作**：总结用户最近更关注什么、表达方式有没有变化、决策逻辑有没有变化。
**输入**：评论区、问大家、用户高频提问、消费反馈
**输出**：用户趋势判断表
**注意**：用户趋势尽量基于真实表达而不是主观感受。

### 阶段4：拆用户关注点变化
**目标**：看用户最近到底在意什么。
**具体动作**：判断用户从"关注什么"转向"更关注什么"。
**输入**：用户高频问题、咨询记录、搜索行为
**输出**：用户关注点变化表
**注意**：重点不是列词，而是看变化方向。

### 阶段5：判断内容趋势
**目标**：看平台最近更愿意放大什么内容。
**具体动作**：判断最近哪些内容形式、内容风格、表达方式更容易被平台和用户接受。
**输入**：平台热点、高表现内容、同行爆款内容
**输出**：内容趋势判断表
**注意**：内容趋势不是"风格新"，而是"效率更高"。

### 阶段6：判断内容形式变化
**目标**：看什么形式更适合当前阶段。
**具体动作**：判断图文、短视频、直播切片、对比测评、口播种草、场景共鸣等形式的效率变化。
**输入**：近期高表现内容样本、平台分发特征
**输出**：内容形式变化表
**注意**：不同赛道适合的内容形式不同。

### 阶段7：判断赛道趋势
**目标**：看哪些细分方向在升温，哪些已经红海。这是整张表最核心的判断。
**具体动作**：判断赛道内哪些方向在上升、哪些方向竞争已非常拥挤、哪些方向适合提前卡位。
**输入**：行业信息、竞品布局、内容趋势、用户趋势
**输出**：赛道趋势判断表

### 阶段8：识别值得追的趋势方向
**目标**：明确哪些方向可以跟进。
**具体动作**：找出至少1个当前值得跟进的趋势方向。
**输入**：用户趋势、内容趋势、赛道趋势结果
**输出**：值得追方向表
**注意**：值得追不代表盲追，要看品牌能不能承接。

### 阶段9：识别需要谨慎跟的热闹方向
**目标**：明确哪些方向表面热闹但未必适合品牌。
**具体动作**：找出至少1个热度很高但竞争拥挤或不适合当前品牌的方向。
**输入**：热点内容、赛道红海方向、平台热点
**输出**：谨慎跟方向表
**注意**：热闹不等于机会，很多方向只是看起来热。

### 阶段10：识别适合提前卡位的方向
**目标**：明确哪些方向值得品牌提前布局。
**具体动作**：找出至少1个当前还没彻底爆，但未来可能值得品牌先占位置的方向。
**输入**：用户趋势、竞品空档、赛道变化
**输出**：提前卡位方向表
**注意**：卡位方向的关键是"现在做不挤，未来有可能放大"。

### 阶段11：判断趋势和品牌匹配度
**目标**：看这些趋势是不是适合自己品牌。
**具体动作**：判断不同趋势方向与品牌能力、品牌调性、供应链、内容能力是否匹配。
**输入**：品牌定位、产品能力、团队资源、平台能力
**输出**：品牌匹配度判断表
**注意**：不是所有趋势都适合自己品牌去追。

### 阶段12：判断趋势和节点匹配度
**目标**：看哪些方向更适合618等节点放大。
**具体动作**：判断哪些趋势方向适合节点种草、节点转化、节点直播预热。
**输入**：节点节奏、活动目标、品类特征
**输出**：节点匹配度判断表
**注意**：同一个趋势方向，不一定适合当下节点。

### 阶段13：输出赛道判断结论
**目标**：把趋势结果沉淀成明确判断。
**具体动作**：用"值得追 / 需谨慎 / 提前卡位"三类方式输出。
**输入**：前面所有趋势结果
**输出**：趋势赛道判断结论表
**注意**：课堂里一定要强行分类，不要只写一堆观察。

### 阶段14：输出首轮动作建议
**目标**：把判断转成动作。
**具体动作**：针对每个方向给出建议：先观察、先测试、先卡位、先投入。
**输入**：趋势结论、品牌资源、时间节点
**输出**：首轮动作建议表

### 阶段15：形成趋势赛道判断表第一版
**目标**：把结果沉淀成标准表。
**具体动作**：将用户趋势、内容趋势、赛道趋势、方向判断、动作建议整合进同一张表。
**输入**：前面所有结果
**输出**：趋势赛道判断表初版（第六张现场输出物）

### 阶段16：建立后续验证机制
**目标**：后续验证趋势判断是否正确。
**具体动作**：定义后续观察信号：用户反应、内容表现、搜索增长、竞品跟进、平台放大量级等。
**输入**：指标口径、内容数据、搜索数据、赛道观察
**输出**：趋势验证指标表
**注意**：趋势判断也要被验证，不是一次性拍脑袋结论。

## 三层判断框架

| 层面 | 回答的问题 | 解释 |
|------|-----------|------|
| 用户趋势 | 用户最近更关注什么 | 看用户需求与表达方式是否在变化 |
| 内容趋势 | 平台最近更愿意放大什么 | 看哪些内容形式和表达方式正在变高效 |
| 赛道趋势 | 哪些细分方向在升温、哪些已红海 | 看行业格局变化和竞争状态 |

## 核心金句

"趋势判断，不是追热点，而是看什么值得追、什么要谨慎、什么适合卡位。"

## 最终输出要求

每个品牌至少判断出：
- 1个值得追的趋势方向
- 1个需要谨慎跟的热闹方向
- 1个适合自己品牌提前卡位的方向
