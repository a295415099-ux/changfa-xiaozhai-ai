## 模板4：课堂交付清单

| 交付物 | 用途 | 备注 |
|-------|------|------|
| 赛道定义表 | 明确这次到底在判断哪个赛道和细分方向 | 不先定义赛道，趋势判断会发散 |
| 用户趋势判断表 | 明确用户最近关注点和决策逻辑的变化 | 趋势判断的第一层基础 |
| 内容趋势判断表 | 明确平台更放大哪些内容形式与表达方式 | 趋势判断的第二层基础 |
| 赛道趋势判断表 | 明确哪些方向升温、哪些红海、哪些可卡位 | 趋势判断的第三层基础 |
| 值得追方向表 | 明确至少1个当前值得跟进的趋势方向 | 帮助团队知道哪些可以主动跟进 |
| 谨慎跟方向表 | 明确至少1个表面热闹但需谨慎的方向 | 避免团队盲目追热闹 |
| 提前卡位方向表 | 明确至少1个适合品牌提前布局的方向 | 帮助品牌建立先发位置 |
| 品牌匹配度判断表 | 明确趋势方向和品牌自身能力是否匹配 | 不是所有趋势都适合自己品牌 |
| 趋势赛道判断表 | 第六张现场输出物的核心表 | 要强行分类成追/谨慎/卡位 |
| 首轮动作建议表 | 明确接下来到底先跟、先避、还是先卡位 | 趋势判断最终要回到动作 |
