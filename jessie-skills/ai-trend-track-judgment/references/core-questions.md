# AI趋势赛道判断 — 核心问题清单

## 第六张表建议回答的关键问题

### 1. 用户趋势：用户最近在变什么？
- 用户最近更关注什么？
- 表达方式有没有变化？
- 决策逻辑有没有变化？

### 2. 内容趋势：平台最近更放大什么？
- 哪些内容形式更容易被放大？
- 哪些内容风格更被接受？
- 哪些表达方式更高效？

### 3. 赛道趋势：方向正在怎么变？
- 哪个细分方向开始升温？
- 哪些赛道已经开始红海？
- 哪些方向更适合618节点放大？

### 4. 最终判断：我们到底要追什么、避什么、卡什么？
至少判断出：
- 1个值得追的方向
- 1个需谨慎跟的方向
- 1个适合提前卡位的方向
