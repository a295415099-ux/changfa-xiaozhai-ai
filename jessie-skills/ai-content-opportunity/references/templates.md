## 3. 机会评分模板

| 内容机会 | 需求强度(1-5) | 竞争空档(1-5) | 趋势热度(1-5) | 品牌承接能力(1-5) | 执行难度(1-5,越低越好) | 综合判断 | 优先级 |
|---------|-------------|-------------|-------------|-----------------|---------------------|---------|-------|
| | | | | | | | |

评分维度说明：
- 需求强度：用户对这个需求的真实性和强烈程度
- 竞争空档：竞品在这个方向的空白程度
- 趋势热度：该方向在平台和用户中的热度
- 品牌承接能力：品牌是否有能力落地该内容方向
- 执行难度：团队落地该机会的难易程度（分数越低越容易）
