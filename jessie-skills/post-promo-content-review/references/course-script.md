# 课用话术参考

## 上屏文案核心

促后复盘不是只看卖得好不好，而是看哪些内容真的有价值、值得复用、值得放大、带来了搜索和成交。

## 六类内容资产（课堂版）

爆款内容 / 高互动内容 / 高转化内容 / 搜索带动内容 / 可投放素材 / 可沉淀关键词。

## 课堂现场动作

1. 找出618期间表现最好的3条内容
2. 找出最值得复用的3个内容结构
3. 找出最值得沉淀的5-10个关键词
4. 找出下半年可以继续放大的1-3类素材

## 课堂输出

《618促后内容资产复盘表》

## 关键词

促后复盘｜爆款内容｜高互动内容｜高转化内容｜搜索带动｜可投素材｜沉淀关键词

## 使用场景提示

在课堂现场演示时：
- 先让学员理解"不是只看销量"的认知转换
- 用六类内容资产框架替换传统的"只看爆款"
- 强调每个阶段的输出都是给团队用的，不是做了就忘
- 最后强调："复盘不是结束，而是下半年继续增长的起点"
