# 小红书笔记模板

> 使用说明：以下模板可直接填写，用于课堂现场或实际笔记生产。

| 模块 | 内容任务 | 填写内容 |
|------|----------|----------|
| 标题 | 吸引目标人群点击 | |
| 开头 | 建立代入感（前3行） | |
| 用户痛点 | 说明为什么需要 | |
| 场景表达 | 放进真实使用场景 | |
| 卖点转译 | 把产品价值讲成人话 | |
| 适合人群 | 降低用户判断成本 | |
| 不适合人群 | 提升可信度 | |
| 关键词 | 帮助用户记忆和搜索 | |
| 评论区引导 | 引导用户互动和提问 | |
| 淘内承接 | 用户后续去哪里看/搜/问 | |
