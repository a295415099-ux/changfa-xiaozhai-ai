# 课程讲稿摘要：小红书笔记生成 Skill

## 标题

Skill 17｜小红书笔记生成 Skill

## 上屏文案核心

小红书不是商品详情页，小红书用户不是来听品牌介绍的，他们更在意真实体验、使用场景、避坑经验、选择建议、情绪共鸣和评论区互动。

## 6个核心结构

标题 / 开头 / 正文主体 / 卖点转译 / 关键词布局 / 评论区引导

## Prompt 输出要求

5个标题方向、开头钩子、正文结构、种草卖点转译、适合人群/不适合人群、关键词布局、评论区互动引导、淘内承接建议

## 本页输出

《小红书笔记模板》

## 现场动作要求

请各组围绕一个核心产品，现场完成一篇可种草、可互动、可承接的小红书笔记结构。

## 适用对象

品牌老板、品牌负责人、内容负责人、小红书运营、电商负责人、客服负责人、培训讲师

## 核心目标

解决小红书内容不能写成产品介绍、官方卖点、活动说明或详情页压缩版，而要写成「一个人，在一个具体场景下，遇到一个真实问题，找到一个可参考的解决方案」

## 课堂一句话总结

小红书种草笔记的核心不是「夸产品」，而是让用户觉得：你懂我的场景，你说中了我的问题，你给了我一个可以参考的选择。
