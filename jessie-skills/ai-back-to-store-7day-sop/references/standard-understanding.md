# 回店7天行动清单 — 标准理解

## 每日核心任务

| 天数 | 核心任务 | 课堂解释 |
|------|----------|----------|
| 第1天 | 统一内部复盘，确定核心方向和优先级 | 先统一方向，不要一回公司就各做各的 |
| 第2天 | 三类词表和表达标准内部再确认 | 让内容、电商、品牌语言重新统一 |
| 第3天 | 店铺承接页、商品页、活动页梳理一轮 | 先把站内接法过一遍 |
| 第4天 | 内容蓄水优先级落到执行清单 | 把"该做什么"变成"今天谁来做什么" |
| 第5天 | 明确搜索承接和店铺优化动作 | 把内容和站内接法串起来 |
| 第6天 | 拉内部协同会，定负责人和节奏 | 让三角色和相关团队真正进入协同状态 |
| 第7天 | 对照课程成果，确认第一轮推进是否已开始 | 不是看懂了没有，而是看有没有真的动起来 |
