# AI搜索回流观察表 SOP 技能包

## 技能名称

**AI搜索回流观察表 SOP（全流程技能）**

## 版本

v1.0 | 2026年6月

## 适用对象

品牌老板、商家操盘手、内容团队、电商团队、直播团队、数据团队、培训讲师

## 技能简介

AI搜索回流观察表SOP全流程技能。用于帮助品牌/商家借助AI观察内容发出去之后，是否开始往搜索回流，帮助团队建立「看回流信号」而不只是「看内容热度」的经营习惯。

## 核心能力

| 维度 | 说明 |
|------|------|
| **品牌搜索观察** | 判断品牌词、品牌+品类词是否被内容带起来 |
| **品类搜索观察** | 判断用户是否从模糊兴趣走向明确搜索 |
| **场景词激活观察** | 判断内容有没有创造新的搜索表达 |
| **承接动作观察** | 判断商品页、店铺页、直播间访问是否提升 |

## 解决的痛点

- 「内容发完就算结束」，不看回流效果
- 「只看播放和点赞，不看搜索回流」
- 「不知道该重点盯哪些回流信号」
- 回流偏弱时不知道怎么定位原因
- 团队没有持续观察搜索回流的习惯

## 全流程概要（17个阶段）

```
阶段0  → 明确本次任务（统一认知：看回流≠看热度）
阶段1  → 锁定核心SKU
阶段2  → 梳理对应内容动作
阶段3  → 提炼观察关键词池
阶段4  → 判断品牌相关搜索信号
阶段5  → 判断品类相关搜索信号
阶段6  → 判断场景词是否被激活
阶段7  → 判断承接动作是否提升
阶段8  → 拆搜索回流层级
阶段9  → 判断哪些信号最值得重点盯
阶段10 → 判断回流是否开始发生
阶段11 → 找出回流偏弱的原因
阶段12 → 输出优化建议
阶段13 → 形成搜索回流观察表第一版
阶段14 → 输出首轮重点观察信号（Top 3-5）
阶段15 → 分配到团队观察动作
阶段16 → 建立持续观察机制
```

## 输出物清单（11项）

1. 核心SKU确认表
2. 内容动作清单
3. 搜索观察关键词池
4. 品牌搜索观察表
5. 品类搜索观察表
6. 场景词激活观察表
7. 承接动作观察表
8. 回流状态判断表
9. **搜索回流观察表（核心输出）**
10. Top信号观察表
11. 团队观察分工表

## 使用方法

1. 将技能包导入 AI 助手（如 JVS Claw）
2. 在需要做搜索回流观察时，触发关键词（如「搜索回流观察」「回流观察表」等）
3. 按照 17 个阶段顺序，结合 9 个 AI 提示词模板逐步推进
4. 配合内置的填写模板完成各阶段输出

## 核心理念

> 蓄水不是做完就结束，还要会判断它有没有开始起作用。这张表的价值：帮助团队建立一个判断习惯——内容发出去之后，到底有没有开始往搜索回流。
