# 优质素材筛选 - 六大评分维度详解

## 评分总则

每条待筛选素材按照以下六个维度逐一打分，每个维度 1-5 分，满分 30 分。
评分必须结合用户提供的实际数据（播放量、互动数据、评论内容、进店数据、加购数据、搜索数据、店铺页面信息等），不可仅凭主观判断。

---

## 1. 种草力（1-5分）

**核心判断问题**：用户是否产生代入和需求？

**评分依据**：
- 评论区是否出现"这说的就是我"、"我也是这样"等代入感表达
- 收藏数是否远高于点赞数（收藏/点赞比 > 0.3 为强信号）
- 是否有用户主动私信询问产品信息
- 完播率是否高于同类内容平均值
- 评论区是否出现"哪里买"、"怎么用"、"适合我吗"等需求信号
- 停留时长是否显著优于同类内容

**分值标准**：
| 分数 | 判断标准 |
|------|----------|
| 5分 | 评论区大量代入表达 + 高收藏/点赞比 + 主动私信 + 完播率高 |
| 4分 | 有明显的代入感评论 + 收藏较高 + 完播高于均值 |
| 3分 | 有一定互动但不是强烈代入，收藏正常水平 |
| 2分 | 播放/点赞有但用户无明显代入和需求表达 |
| 1分 | 只有数据指标没有真实种草互动信号 |

---

## 2. 关键词力（1-5分）

**核心判断问题**：是否强化搜索词/卖点词/场景词？

**评分依据**：
- 标题中是否包含品牌核心搜索词或品类词
- 口播/文案中是否自然植入关键搜索词
- 评论区是否有用户讨论品牌/品类关键词
- 是否引导用户搜索了相关关键词（搜索数据可见）
- 商品标题与素材关键词的匹配度
- 是否区分了搜索词、卖点词、场景词三类关键词

**分值标准**：
| 分数 | 判断标准 |
|------|----------|
| 5分 | 标题含搜索词 + 口播自然植词 + 评论区讨论关键词 + 搜索数据验证 |
| 4分 | 关键词布局较完整，搜索词/卖点词/场景词覆盖两类以上 |
| 3分 | 有关键词意识但覆盖不够全面 |
| 2分 | 仅标题含部分关键词，口播和评论区无强化 |
| 1分 | 内容完全无关键词布局 |

---

## 3. 转化力（1-5分）

**核心判断问题**：是否带来点击、进店、加购、咨询？

**评分依据**：
- 点击率（CTR）是否高于同品类均值
- 商品卡点击量
- 进店UV及进店率
- 加购数及加购率
- 咨询/客服对话发起量
- 成交辅助（素材辅助成交的归因数据）

**分值标准**：
| 分数 | 判断标准 |
|------|----------|
| 5分 | CTR高 + 进店率高 + 加购率高 + 有明显成交辅助数据 |
| 4分 | 进店和加购明显优于均值 |
| 3分 | 有一定进店但转化效率一般 |
| 2分 | 有曝光和点击但进店率低 |
| 1分 | 播放高但不转化，无进店数据 |

---

## 4. 互动质量（1-5分）

**核心判断问题**：评论区是否出现真实购买问题？

**评分依据**：
- 是否出现"适合谁"类型提问（人群匹配）
- 是否出现"怎么选"类型提问（决策辅助）
- 是否出现"搜什么"类型提问（搜索引导）
- 是否出现"和什么区别"类型提问（竞品对比）
- 是否出现"有没有直播间"类型提问（直播引导）
- 泛互动（纯表情、"好看"、"点赞"）不算高质量互动

**分值标准**：
| 分数 | 判断标准 |
|------|----------|
| 5分 | 评论区3类以上真实购买问题 + 大量有价值的消费决策讨论 |
| 4分 | 出现2类以上购买问题 + 有一定消费决策讨论 |
| 3分 | 有1-2类购买问题但以泛互动为主 |
| 2分 | 有互动但基本为泛互动，无购买决策讨论 |
| 1分 | 评论区活跃但全部是泛互动，无购买意向信号 |

---

## 5. 承接匹配（1-5分）

**核心判断问题**：店铺页面是否能接住这条内容？

**评分依据**：
- 商品标题是否包含素材中强调的关键词
- 主图是否呼应素材中的种草点
- 详情页首屏是否与素材的种草理由一致
- 客服话术是否准备好回答素材评论区常见问题
- 直播间话术是否承接素材引流的用户
- 素材→搜索→页面的语言一致性

**分值标准**：
| 分数 | 判断标准 |
|------|----------|
| 5分 | 商品标题/主图/详情页首屏/客服/直播全链路承接 |
| 4分 | 商品页承接较好，客服和直播间基本到位 |
| 3分 | 商品页部分承接，客服和直播待补充 |
| 2分 | 商品页与素材关联弱，承接不充分 |
| 1分 | 店铺页面完全无法承接素材流量 |

---

## 6. 复用价值（1-5分）

**核心判断问题**：是否能被改写、二创、投放、沉淀？

**评分依据**：
- 是否可改成短视频（不同时长/不同开头）
- 是否可改成小红书笔记（图文形式）
- 是否可作为直播口播素材
- 是否可提取为详情页模块
- 是否可沉淀为客服话术/FAQ
- 是否可作为达人Brief模板
- 是否可作为投放素材（信息流、搜索广告等）

**分值标准**：
| 分数 | 判断标准 |
|------|----------|
| 5分 | 可覆盖5种以上复用形式，结构清晰可拆解 |
| 4分 | 可覆盖3-4种复用形式 |
| 3分 | 可覆盖2种复用形式 |
| 2分 | 仅可复用为1种形式 |
| 1分 | 内容不可拆解、不可复用 |
