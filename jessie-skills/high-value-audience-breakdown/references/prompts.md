# AI提示词模板（高价值人群拆解）

本文档包含各关键步骤对应的完整AI提示词。使用时将 `____` 替换为实际内容。

---

## 阶段 0：明确本次任务

### 开场确认提示词

```
我们现在要做一个非常重要的工作：把品牌的泛人群标签升级成可执行的高价值人群画像。

先说清楚本次任务的背景：
- 品牌名：____
- 当前主推产品/品类：____
- 当前最想解决的内容问题：____（例如：不知道该对谁写内容 / 内容太泛没人看 / 有内容没转化）

请确认以上信息是否准确，我再开始拆解。
```

---

## 阶段 1：回收基础资料

### 资料收集引导提示词

```
为了拆出真实可用的高价值人群，我需要你先帮我回收一些基础资料。
不要求一次性给齐，有多少给多少就行：

📋 请尽量提供：
1. 【用户画像】现有成交用户的年龄、性别、城市、消费力等
2. 【成交数据】订单情况、复购率、客单价大致分布
3. 【用户反馈】评论区截取一些典型评价、客服问答中的高频问题
4. 【商品矩阵】商品清单，哪些是主推款、引流款、利润款、形象款
5. 【内容表现】各平台做得好的内容链接或截图，以及互动数据
6. 【搜索数据】品牌词、品类词、竞品词搜索情况（有的话）

你先把有的发给我，我来整理。
```

### 资料整理后的确认提示词

```
我把你提供的资料整理成了「人群基础资料表」，请确认：
- 有无遗漏重要信息？
- 有无需要补充的用户群体？
- 数据范围是否合理？

确认后我们进入下一步。
```

## 阶段 2：统一拆解原则

### 判断口径建立提示词

```
在开始拆解之前，我们先对齐一个关键认知：

❌ 高价值人群 ≠ 最大的人群
✅ 高价值人群 = 最容易被种草 + 最容易产生需求 + 最容易被说服 + 最容易形成成交 + 最值得持续经营

请帮我围绕【____品牌】的主推产品【____】，逐一评估以下五个维度中哪些人群特征更符合「高价值」：

1. 种草触发度：看到什么内容最容易被触动？
2. 需求匹配度：哪类人群的需求与品牌产品最匹配？
3. 说服容易度：哪类人更容易相信品牌的证据（成分/测评/案例/口碑）？
4. 成交转化率：哪类人的下单决策路径最短？
5. 长期经营价值：哪类人买了之后还能持续复购/推荐？

输出一个「高价值人群判断口径表」，作为后面拆解的标尺。
```

## 阶段 3：拆解新客人群

```
请帮我拆解【____品牌】的新客人群。

新客人群定义：对品牌和品类还不熟，但对问题或场景已有初步兴趣的潜在用户。

请围绕以下维度输出：
1. 用户是谁：年龄、性别、城市、生活方式、一句话人设
2. 核心痛点：他们有什么问题但还没找品牌解决？
3. 首次触达场景：在什么情况下会第一次注意到品牌？
4. 入门购买动机：为什么愿意第一次试试品牌？
5. 决策阻力：什么让他们犹豫不下单？价格？信任？品类认知？
6. 内容触发点：什么样的内容（标题/封面/话题）能让他们停下来看？
7. 对应产品：哪些入门款/体验款最适合承接新客？

请使用你之前整理的品牌资料作为依据，不要凭空推测。如果某个维度信息不足，请标注"需补充__资料"。
```

---

## 阶段 4：拆解功效/需求人群

```
请帮我拆解【____品牌】的功效/需求人群。

功效人群定义：已经被某个问题困扰一段时间，正在主动搜索解决方案的精准需求用户。

请围绕以下维度输出：
1. 用户是谁：年龄、性别、城市、生活方式
2. 核心功效/需求诉求：他们最想要解决什么具体问题？
3. 问题严重程度：是轻度困扰还是严重影响生活/工作/状态？
4. 搜索行为：他们用哪些关键词在哪些平台搜索？（请根据搜索数据合理推断）
5. 对比维度：他们在对比品牌时关注什么？（效果/成分/价格/口碑/安全性）
6. 证据需求：什么证据能让他们相信品牌？（成分解析/对比测评/真实案例/数据）
7. 内容触发点：什么样的功效内容能打动他们？
8. 对应产品：哪些功效型产品最适合他们？

请基于品牌已有资料拆解，不足处标注说明。
```

---

## 阶段 5：拆解场景人群

```
请帮我拆解【____品牌】的场景人群。

场景人群定义：平时不一定有需求，但在特定场景中被激发出购买欲望的人群。

请围绕以下维度输出：
1. 用户是谁：他们平时不觉得需要这个产品，但在什么角色/状态下会被触发？
2. 高价值场景清单：列出与品牌相关的高频/中频/低频场景（至少5个场景）
   - 示例场景：通勤/旅行/熬夜/换季/露营/送礼/聚会/约会/考试/面试/搬家
3. 每个场景的动机：这个场景激发了什么需求/情绪？
4. 场景情绪：焦虑？期待？社交压力？自我奖赏？仪式感？
5. 内容触发点：什么样的场景内容能让用户产生"这不就是我吗"的代入感？
6. 对应产品：什么产品最匹配这个场景？

注意：场景人群的关键不是列出场景名字，而是让用户产生代入感。
```

---

## 阶段 6：拆解高复购人群

```
请帮我拆解【____品牌】的高复购人群。

高复购人群定义：已经完成首次购买、有复购潜力的老客群体。

请围绕以下维度输出：
1. 用户是谁：他们购买前是什么状态？买后的变化是什么？
2. 首次购买行为：买了什么？为什么买？什么内容/渠道带来的？
3. 复购触发点：什么情况下会再次购买？（用完了/效果很好/活动/新品）
4. 二次需求：除了复购同款，还有哪些升级/搭配/替旧需求？
5. 升单路径：从入门款到进阶款到高客单款的路径是什么？
6. 推荐意愿：什么情况下会推荐给朋友？
7. 内容触发点：什么样的内容能触发他们复购或升单？（老客回访/升级对比/组合推荐）
8. 对应产品：进阶款/组合装/会员权益/订阅制

高复购人群是长期经营价值最高的一类，请重点拆解。
```

---

## 阶段 7：拆解价格敏感人群

```
请帮我拆解【____品牌】的价格敏感人群。

价格敏感人群定义：有需求但价格是核心决策变量的人群——不是没钱，而是对性价比要求更高。

请围绕以下维度输出：
1. 用户是谁：他们是真没钱还是对品类价值认知不够？（这两种策略完全不同）
2. 价格敏感原因：品类认知低？竞品价格更低？消费习惯？观望等大促？
3. 触发下单的条件：大促折扣？组合装算下来更划算？赠品价值？限时机制？
4. 合理价格带：他们愿意接受的价格范围是多少？
5. 内容触发点：什么内容能让他们觉得"现在买很划算"？
6. 对应产品：引流款/组合装/活动款/试用款

重点不是低价，而是降低决策阻力——组合装让单价更低、赠品提高价值感、限时制造紧迫感。
```

## 阶段 8：补充人群细节

```
现在五类人群画像已经有了框架，请帮我补充以下细节，让画像真正能用于内容创作：

对每一个人群类型，补充：
1. 一句人话描述：让内容团队一看就懂这个人群是谁（不要用专业术语）
   示例：不是"25-35岁都市白领女性"，而是"加班到10点回家，累到不想卸妆但知道必须卸的女生"
2. 典型用户故事：虚构一个2-3句话的典型用户场景（谁/在什么时间地点/遇到什么问题/想要什么）
3. 关键词库：
   - 搜索词：他们会搜什么
   - 痛点词：他们怎么描述自己的问题
   - 场景词：他们在什么场景下想起你
   - 情绪词：他们的情绪是怎样的
4. 内容调性建议：这类人喜欢什么语言风格？严肃专业/轻松幽默/温暖治愈/直接利落？

输出「高价值人群画像表草稿」。
```

## 阶段 9：判断人群优先级

```
现在五类人群画像已经完成，请帮我判断当前阶段最该优先做谁。

请从以下维度逐一评估：

1.【品类阶段】
- 新品类/新品牌 → 优先新客 + 功效人群
- 成熟品类/老品牌 → 优先场景 + 复购人群
- 品牌当前阶段是：____

2.【商品结构】
- 有强力引流款 → 可以优先打新客
- 有成熟复购款 → 可以优先打老客
- 当前商品矩阵主力是：____

3.【内容能力】
- 团队擅长做测评/成分/科普 → 优先功效人群
- 擅长场景种草/生活方式 → 优先场景人群
- 擅长活动策划/直播 → 优先价格敏感+复购人群

4.【转化效率】
- 哪类人群从内容→成交路径最短？

5.【竞争环境】
- 哪类人群在竞争中相对空白？

请综合以上维度，给五类人群标注优先级（高/中/低），并说明理由。输出「5类高价值人群优先级表」。

最多明确2-3类高优先级人群作为当前阶段重点。
```

## 阶段 10：生成内容触发点

```
现在请为每类高价值人群生成可直接用于内容创作的内容触发点方案：

对每一类人群，输出：
1. 内容触发点一句话：用户看到什么会被触发需求？
2. 内容方向建议：
   - 适合的内容形式（笔记/视频/图文/直播）
   - 适合的内容主题方向
3. 说服逻辑：这篇内容凭什么让他信？
4. CTA引导：看完内容后引导他做什么？（搜索品牌词/点进直播间/收藏加购/直接下单）
5. 参考标题示例：给3-5个可直接使用或改编的标题方向

输出「人群对应内容触发点表」。
```

## 阶段 11：生成人群—产品匹配

```
请帮我把高价值人群与商品矩阵做精确对应：

对每一类人群，明确：
1. 适合推的产品：具体到产品名/SPU
2. 产品组合建议：是否推组合装？买A送B？
3. 推荐优先级：主力推荐/次推/尝试推荐
4. 匹配理由：为什么这个产品适合这类人群？
5. 注意避免的匹配：哪些产品不该推给这类人群？（防止推荐错位）

输出「人群—产品匹配表」。

规则参考：
- 新人 → 入门款/体验款/小规格
- 功效需求 → 功效款/明星单品
- 场景触发 → 场景款/礼盒/限定款
- 复购 → 进阶款/组合装/会员款
- 价格敏感 → 引流款/活动组合装/大促款
```

```
请把前面所有拆解结果整合成一份完整的「高价值人群画像表」。

表头结构：
| 人群类型 | 用户是谁 | 核心痛点 | 购买动机 | 内容触发点 | 对应产品 | 优先级 |
|---------|---------|---------|---------|-----------|---------|--------|
| 新客人群 |         |         |         |           |         | 高/中/低 |
| 功效人群 |         |         |         |           |         | 高/中/低 |
| 场景人群 |         |         |         |           |         | 高/中/低 |
| 高复购人群 |       |         |         |           |         | 高/中/低 |
| 价格敏感人群 |     |         |         |           |         | 高/中/低 |

要求：
- 「用户是谁」字段使用一句人话描述，不要泛标签
- 每个字段的内容控制在2-3句话以内，但信息量要足够
- 标注数据来源（基于真实数据/基于合理推断/待验证）

这是本 Skill 的核心正式输出，请确保完整、可执行。
```

## 阶段 13：输出优先种草人群

```
请从画像表中筛出当前阶段最值得优先种草的 1-3 类人群，生成「下阶段优先种草人群清单」。

清单格式：
1. 优先人群：____ ｜ 优先级：高 ｜ 为什么先做这个人群：____
2. 3个月内对应主推产品：____
3. 建议内容打法方向：____
4. 建议投放定向方向：____
5. 建议电商承接动作：____

这份清单的目的是：让内容团队在接下来3个月不再回到泛人群表达，而是聚焦在明确画像上创作。
```

## 阶段 14：团队使用分工

```
画像表不能只停留在方案里，必须分配到每个团队去使用。请输出「团队使用分工表」：

| 团队 | 使用人群画像的重点 | 具体动作 | 注意事项 |
|------|-----------------|---------|---------|
| 内容团队 | 按画像写内容 | 选题/标题/正文/封面都对齐画像 | 不要写泛内容 |
| 电商团队 | 按人群做承接 | 详情页/直播间话术/店铺推荐 | 不同人看到不同产品 |
| 投放团队 | 按画像定向 | 人群包/素材方向/出价策略 | 新客和老客素材要分开 |
| 客服团队 | 按人群沟通 | 接待话术/推荐逻辑/异议处理 | 不同类型人群的犹豫点不同 |
| 私域团队 | 按画像分层 | 会员分层/回访机制/老客推荐 | 高复购人群重点经营 |
```

## 阶段 15：建立更新机制

```
高价值人群不是固定不变的，会随着新品、活动、用户反馈、竞争变化而更新。

请设计「人群画像更新机制表」：

| 更新触发条件 | 更新什么 | 更新频率 | 负责人建议 |
|------------|---------|---------|-----------|
| 新品上线 | 更新对应人群和产品匹配 | 新品上时 | 商品+内容 |
| 大促节点 | 重新评估价格敏感人群 | 大促前 | 电商+投放 |
| 评论区新反馈 | 补充痛点与购买动机 | 月度 | 客服+内容 |
| 搜索词变化 | 更新搜索行为和需求描述 | 月度 | 电商+内容 |
| 竞品人群变化 | 关注竞品在打哪类人群 | 季度 | 内容+投放 |
| 季度复盘 | 系统更新全部画像表 | 季度 | 全团队 |

画像不是课堂作业，是活文档。
```
