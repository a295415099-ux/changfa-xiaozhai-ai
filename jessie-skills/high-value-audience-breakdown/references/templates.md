## 二、高价值人群判断口径表

| 评估维度 | 高价值特征描述 | 低价值特征（避免） |
|---------|--------------|-----------------|
| 种草触发度 | 看到某一类内容容易被触动 | 内容无感，泛泛划过 |
| 需求匹配度 | 需求与品牌产品高度匹配 | 需求模糊或不对口 |
| 说服容易度 | 容易相信证据（成分/测评/案例/口碑） | 极难说服，需要反复教育 |
| 成交转化率 | 从内容到下单路径短 | 要反复触达多次才可能转化 |
| 长期经营价值 | 复购潜力大/推荐意愿强 | 一锤子买卖或无复购需求 |

## 三、五类人群拆解框架

### 3.1 新客人群拆解框架

| 维度 | 内容 |
|------|------|
| 用户是谁 | 年龄/性别/城市/生活方式/一句话人设 |
| 核心痛点 | 有什么问题但还没找品牌解决 |
| 首次触达场景 | 在什么情况下会第一次注意到品牌 |
| 入门购买动机 | 为什么愿意第一次试试 |
| 决策阻力 | 什么让他们犹豫不下单（价格/信任/品类认知） |
| 内容触发点 | 什么内容能让他们停下来看 |
| 对应产品 | 入门款/体验款/小规格 |

### 3.2 功效/需求人群拆解框架

| 维度 | 内容 |
|------|------|
| 用户是谁 | 年龄/性别/城市/生活方式 |
| 核心功效诉求 | 最想解决什么具体问题 |
| 问题严重程度 | 轻度困扰/中度/严重影响 |
| 搜索行为 | 搜索关键词 + 搜索平台 |
| 对比维度 | 在比什么（效果/成分/价格/口碑/安全） |
| 证据需求 | 什么证据能让他们信 |
| 内容触发点 | 什么样的功效内容能打动 |
| 对应产品 | 功效款/明星单品 |

### 3.3 场景人群拆解框架

| 维度 | 内容 |
|------|------|
| 用户是谁 | 在什么角色/状态下被触发 |
| 场景清单 | 5+个场景（标注频率：高频/中频/低频） |
| 场景动机 | 每个场景激发了什么需求/情绪 |
| 场景情绪 | 焦虑/期待/社交压力/自我奖赏/仪式感 |
| 内容触发点 | 什么场景内容能产生代入感 |
| 对应产品 | 场景款/礼盒/限定款 |

### 3.4 高复购人群拆解框架

| 维度 | 内容 |
|------|------|
| 用户是谁 | 购买前状态 + 购买后变化 |
| 首次购买 | 买了什么/为什么买/什么渠道带来的 |
| 复购触发点 | 用完了/效果很好/活动/新品/推荐 |
| 二次需求 | 升级需求/搭配需求/替旧需求 |
| 升单路径 | 入门→进阶→高客单的路径 |
| 推荐意愿 | 什么情况下推荐给朋友 |
| 内容触发点 | 什么内容触发复购/升单 |
| 对应产品 | 进阶款/组合装/会员权益 |

### 3.5 价格敏感人群拆解框架

| 维度 | 内容 |
|------|------|
| 用户是谁 | 真预算有限 vs 品类认知不够 |
| 敏感原因 | 认知低/竞品低价/消费习惯/等大促 |
| 触发下单条件 | 大促/组合装划算/赠品/限时 |
| 合理价格带 | 他们愿意接受的价格范围 |
| 内容触发点 | 什么内容让他们觉得"现在买很划算" |
| 对应产品 | 引流款/组合装/活动款 |

## 四、画像补充维度

在阶段8中为每个人群补充：

### 4.1 一句人话描述

格式：不是「标签式描述」，而是「具体人在具体场景中的一句话概括」

❌ 错误：25-35岁都市白领女性
✅ 正确：加班到10点回家，累到不想卸妆但知道必须卸的女生

### 4.2 典型用户故事

格式：谁 / 在什么时间地点 / 遇到什么问题 / 想要什么结果（2-3句话）

### 4.3 关键词库

| 关键词类型 | 示例 |
|-----------|------|
| 搜索词 | 他们会用什么词在平台搜索 |
| 痛点词 | 怎么描述自己的问题和困扰 |
| 场景词 | 在什么场景下会想起品牌 |
| 情绪词 | 描述情绪的关键词（焦虑/期待/疲惫/兴奋） |

### 4.4 内容调性建议

为该人群选择语言风格：严肃专业 / 轻松幽默 / 温暖治愈 / 直接利落 / 生活方式感

## 五、优先级判断维度

五类人群优先级标注（高/中/低），评估维度：

| 维度 | 高分特征 | 低分特征 |
|------|---------|---------|
| 品类阶段 | 新品类打新客+功效；成熟品类打场景+复购 | 逆品类阶段 |
| 商品结构 | 有对应产品矩阵可承接 | 缺对应产品 |
| 内容能力 | 团队擅长做这类内容 | 团队不擅长 |
| 转化效率 | 内容到成交路径短 | 需要多轮触达 |
| 竞争环境 | 竞争相对空白 | 竞争激烈已红海 |
