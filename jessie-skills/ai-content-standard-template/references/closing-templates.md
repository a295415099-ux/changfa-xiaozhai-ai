# 结尾模板参考

## 五种结尾导向详解

### 1. 导搜索
**适用场景**：需要用户主动搜索才能承接的内容
**公式**：总结价值 → 给出具体搜索词 → 描述搜到后的体验

```
"如果你也想[获得某效果]，可以去[平台]搜「[品牌词+品类词/卖点词]」，
你会发现[搜到后的价值/体验]"
```

### 2. 导店
**适用场景**：店铺有明确承接活动
**公式**：内容铺垫 → 引向店铺 → 说明进店能得到什么

```
"想进一步了解的话，可以去[品牌名]店铺看看。
最近有[活动/优惠/新品]，[到店价值]"
```

### 3. 导商品
**适用场景**：单一爆品驱动，可直接转化
**公式**：强调卖点 → 引向商品 → 明确行动

```
"这个[SKU]我已经用了[时间]，真的[核心感受]。
链接放在[位置]，需要的朋友可以直接看"
```

### 4. 导直播
**适用场景**：直播是主要成交场
**公式**：内容种草 → 预告直播 → 给出时间+看什么

```
"如果想看更多[品类]的实测/对比，我每周[时间]在[平台]直播，
到时候会[播什么]，来直播间可以直接问"
```

### 5. 导私域
**适用场景**：需要长期关系维护的品类
**公式**：提供长期价值 → 引导加私域 → 说明私域能得到什么

```
"关于[话题]，其实还有很多细节，一篇讲不完。
对[话题]感兴趣的朋友，可以[加私域方式]，
我会定期分享[价值内容]"
```

## 结尾黄金法则

1. **一个内容只导一个方向** — 不要同时导搜索+导店+导私域
2. **结尾一定和后链路动作挂钩** — 不能只做一句"欢迎咨询"
3. **给出具体动作而非模糊邀请** — "搜XX词" > "来了解我们"
4. **结尾和开头形成闭环** — 开头抛问题，结尾给搜索路径

## 常见问题

- ❌ "欢迎咨询""评论区见" — 太软，没给明确动作
- ❌ 一个结尾导三个方向 — 用户不知道该做哪个
- ❌ 导了搜索但没给具体搜索词
- ❌ 导了私域但没给加私域的理由
