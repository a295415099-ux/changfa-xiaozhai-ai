# 开头模板参考

## 开头公式

好的内容开头 = **抓力元素** + **场景/问题共鸣** + **自然过渡到SKU**

## 各类型开头模板

### 场景型开头模板
```
[具体时间/地点/情境]，[人物]正在[做什么]。
[描述一个让人产生共鸣的细节]。
[自然过渡]：这也是为什么我推荐[SKU]的原因。
```

### 问题型开头模板
```
你是不是也这样：[列出1-2个具体的问题表现]？
其实核心问题是[本质]。
[解决方向]：用[SKU]，因为[一句话核心卖点]。
```

### 冲突型开头模板
```
你知道[常见认知/做法]吗？
但今天我们想说的是：这个认知可能需要更新了。
因为[新发现/新做法]，所以[SKU]是这样做的...
```

### 结果型开头模板
```
先说结果：[量化的效果/变化]。
怎么做到的？核心就是[关键点]。
用到的东西很简单：[SKU] + [简单方法]。
```

## 小红书平台开头适配

小红书开头强调"收藏感"和"实用价值"：
- 标题加入数字、emoji、标签化表达
- 开头第一句要像"对闺蜜说话"
- 封面+标题+开头第一句必须形成有效信息链

## 抖音平台开头适配

抖音开头直接"黄金3秒"：
- 前3秒必须有信息量或情绪钩子
- 开头可用"反问""数据""反常识"增强吸引力
- 建议先用一句话讲结果或冲突

## 常见问题

- ❌ 开头太长，3秒还没入正题
- ❌ 上来就列卖点，缺少钩子
- ❌ 开头和正文风格不统一
- ❌ 照搬其他品牌的固定开头模板
