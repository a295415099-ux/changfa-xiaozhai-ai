# 搜索词植入模板参考

## 品牌词植入模板

```
在内容中间自然带出品牌名，不要刻意强调：
"...这也是为什么我一直选[品牌名]的[品类]..."
"...[品牌名]在这件事上的思路是..."
```

## 品类词植入模板

```
品类词要在用户有需求感知时自然出现：
"如果你也在找[品类词]..."
"市面上的[品类词]很多，但..."
"选[品类词]有一个关键点..."
```

## 卖点词植入模板

```
卖点词是搜索意图的核心，需要和品类词组合：
"找[品类词]，最重要的其实是[卖点词]"
"[卖点词]的[品类词]和普通的差别在哪..."
```

## 场景词植入模板

```
场景词放在开头或场景描述中：
"[时间]出门前/约会前/上班前..."
"作为一个[人群标签]，我试过很多..."
```

## 结尾搜索引导模板

```
不要说"搜我们"，给出具体可搜的词：
"想了解更多，可以在[平台]搜「[具体搜索词]」"
"下次选购[品类]的时候，记得搜「[品牌词+卖点词]」"
```

## 平台差异注意

- **小红书**：搜索词可以放在笔记结尾的自然引导中
- **抖音**：评论区置顶搜索引导效果更好
- **视频号**：搜索引导要更"软"，不要太硬广
- **站内**：搜索词直接影响站内搜索权重
