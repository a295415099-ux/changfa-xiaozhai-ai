# AI推荐内收链路检查表 SOP

> 未来新增的不只是外种内收，而是AI推荐内收。
> AI已经是用户进店前第二决策入口。

## 用途

检查品牌从内容种草到AI推荐、再到店铺成交的链路是否断裂。

## 最终输出

- 《AI推荐内收链路检查表》
- 《外种—AI问答—进店承接优化清单》
- 《7天链路修复行动表》
- 《链路断点优先级表》
- 《团队执行分工表》

## 检查模块（8个）

1. 外部内容是否清楚
2. 用户问题是否覆盖
3. 商品信息是否一致
4. 推荐理由是否明确
5. 信任证据是否充足
6. 承接页面是否匹配
7. 客服是否能接住
8. 链路闭环检查

## 执行步骤（17步）

**阶段0 明确任务**
- 步骤0: 统一认知，确认AI是第二决策入口

**阶段1 回收资料**
- 步骤1: 回收内容/页面/客服/证据四类资料

**阶段2-8 逐模块检查**
- 步骤2: 外部内容清晰度 → 步骤8: 客服承接

**阶段9 梳理断点**
- 步骤9: 提炼主断点+2-3个次断点

**阶段10-11 生成计划**
- 步骤10: 优化动作清单 → 步骤11: 7天修复计划

**阶段12-14 输出成果**
- 步骤12: 闭环检查 → 步骤13: 正式检查表 → 步骤14: 关键缺口清单

**阶段15-16 执行复盘**
- 步骤15: 分配团队 → 步骤16: 滚动复盘机制
