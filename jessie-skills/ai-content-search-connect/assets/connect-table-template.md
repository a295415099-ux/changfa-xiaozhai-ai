# 内容—搜索—承接连接表（课堂现场可直接填写）

## 连接表填写说明

将"内容—搜索—承接"三段整合成一条标准链路。按以下13列逐一填写：

| 列号 | 列名 | 填写说明 |
|------|------|----------|
| 1 | 核心SKU | 本次分析的主要产品SKU |
| 2 | 内容名称/编号 | 内容的唯一标识名称或编号 |
| 3 | 内容形式 | 短视频/图文/达人内容/直播切片/其他 |
| 4 | 卖点 | 内容在强调哪个核心卖点 |
| 5 | 场景 | 内容面向什么使用场景 |
| 6 | 目标人群 | 内容针对哪类目标人群 |
| 7 | 内容目的 | 种草/解释/搜索引导/承接转化/直播引流 |
| 8 | 用户看完后会搜什么 | 推演的用户搜索词 |
| 9 | 搜索词分类 | 品类词/品牌词/卖点词/场景词/需求词 |
| 10 | 承接落点 | 商品页/店铺页/活动页/直播间/私域 |
| 11 | 承接页链接/入口 | 具体落地页URL或入口描述 |
| 12 | 语言是否一致 | 是/否/部分一致 |
| 13 | 备注/待补动作 | 需要后续补充的优化动作 |
