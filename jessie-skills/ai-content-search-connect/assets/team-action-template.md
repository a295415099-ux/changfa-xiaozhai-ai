# 团队动作拆解模板

## 使用说明

将连接表的断点和补链路建议，按团队拆解为具体可执行的动作。

## 空白模板

| 团队 | 当前问题 | 要补的动作 | 负责人/时间 | 优先级 |
|------|---------|-----------|------------|--------|
| 内容团队 |         |           |            |        |
| 电商团队 |         |           |            |        |
| 直播团队 |         |           |            |        |
| 客服团队 |         |           |            |        |
| 设计/页面团队 |         |           |            |        |

## 优先级标记建议

- 🔴 高优先：链路核心断裂点，不补则整条路径无效
- 🟡 中优先：影响效率但不阻断链路
- 🟢 低优先：锦上添花的优化项
