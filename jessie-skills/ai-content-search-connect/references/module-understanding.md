# 模块理解：内容—搜索—承接的标准定义

## 三段模块核心理念

| 模块 | 它负责什么 | 如果缺失会怎样 |
|------|-----------|---------------|
| 内容 | 激发兴趣、种卖点、种场景、种关键词 | 用户看完没有下一步动作 |
| 搜索 | 把模糊兴趣变成明确动作，让用户搜得到 | 用户被打动了，但不知道怎么找你 |
| 承接 | 商品页讲透、店铺页建立信任、活动页放大利益点、直播间完成解释和转化 | 用户搜到了，但没被接住，也不会成交 |

## 课程讲稿逻辑（用于教学/演示场景）

**标题**：一条真正能为618（大促）服务的内容，不只负责被看见，还要负责把搜索和承接串起来。

**开场逻辑**：很多品牌内容最大的问题，不是不被看见，而是"看见之后没有下一步"。所以内容不是上游动作，而是整条增长链路的起点。

**内容负责什么**：激发兴趣 / 种卖点 / 种场景 / 种关键词

**搜索负责什么**：把模糊兴趣变成明确动作 / 让用户搜得到 / 让用户搜出来后看到的是对的表达

**承接负责什么**：商品页讲透 / 店铺页建立信任 / 活动页放大利益点 / 直播间完成解释和转化

**金句**：
- 内容不是起点，内容是链路的第一环。
- 《内容—搜索—承接连接表》决定品牌的地图能不能真正落地。如果这张表补不出来，说明现在还只是"内容创意图"，还不是"增长链路图"。

**现场动作要求**：每个品牌小组从核心SKU里，至少拿出1条最重要的内容路径，先填出《内容—搜索—承接连接表》。
