# 核心问题清单：连接表要回答的关键问题

在执行SOP流程时，始终围绕以下4个核心问题来推进：

| 问题 | 展开说明 |
|------|----------|
| 1. 这条内容在讲什么？ | 哪个卖点 / 哪个场景 / 哪类人群 / 哪种内容目的 |
| 2. 用户看完后会搜什么？ | 品类词 / 品牌词 / 卖点词 / 场景词 / 需求词 |
| 3. 用户搜到之后落在哪里？ | 商品页 / 店铺页 / 活动页 / 直播间 / 私域 |
| 4. 承接页是否和内容是一套语言？ | 如果不是，这一条链路就已经断了 |

## 使用方式

每个阶段执行完毕后，用这4个问题做一次自检：
- 如果某个问题回答不上来 → 说明该环节还没分析到位
- 如果4个问题都能回答清楚 → 连接表可以出初版
