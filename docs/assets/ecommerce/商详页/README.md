# 商详页视觉资产库

这里统一归档商品详情页长图、首屏、卖点模块、信任证据、评价区和问大家等页面素材。总控台顶部点击 `商详页`，即可按产品、平台和版本浏览。

## 文件夹结构

```text
assets/ecommerce/商详页/{产品}/{平台}/{版本}/
```

例如：

```text
assets/ecommerce/商详页/虫草首乌发膜/天猫/V03_0717上线版/
```

版本建议按 `V序号_日期_阶段` 命名，例如 `V01_0627初稿`、`V02_0703视觉文案改版`、`V03_0717上线版`。

## 每个版本放什么

- 必需：完整商详长图，或按页面顺序编号的模块图。
- 可选：`_版本说明.json`，记录本轮假设、视觉和文案改动。
- 可选：`_数据.csv`，记录上线前、上线后和变化值。

图片支持 `jpg`、`jpeg`、`png`、`webp`、`gif`、`svg`，视频支持 `mp4`、`mov`、`webm`、`m4v`。长图在总控台内按原比例全宽阅读，点击可进入全屏查看。

## 建议复盘指标

- 页面停留、跳失、首屏到达和模块点击。
- 收藏、加购、支付转化和客单价。
- 同一产品不同版本的转化率与关键模块表现变化。

历史项目素材已经通过 `assets/ecommerce/视觉资产索引.json` 接入，不需要立刻搬动；后续新增版本按上述文件夹结构放入即可。
