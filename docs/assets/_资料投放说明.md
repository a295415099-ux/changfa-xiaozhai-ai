# 长发小寨资料库投放说明

## 目录用途

| 目录 | 放什么 |
|---|---|
| `brand/` | 品牌定位、VI、LOGO、色彩、字体、品牌故事、品牌规范 |
| `sku/` | 单品资料、产品图、卖点、规格、价格、用户评价、客服问题 |
| `ecommerce/` | 店铺首页、主图、详情页、活动页、投放素材、电商数据 |
| `social/` | 小红书、抖音、视频号、公众号、私域内容与数据 |
| `packaging/` | 包装图、包装源文件、尺寸、材质、打样、供应链要求 |
| `data/` | 搜索、投放、成交、转化、复购、内容表现等数据表 |
| `reviews/` | 用户评价、问大家、客服聊天、达人反馈、社媒评论 |
| `references/` | 竞品、行业案例、设计参考、课程资料、外部链接 |

## 建议命名

文件名尽量包含：日期、品牌/产品、渠道、用途。

示例：

- `2026-06-核心SKUA-详情页截图.png`
- `2026-06-小红书爆文评论整理.xlsx`
- `2026-06-新品包装打样反馈.md`
- `2026-06-品牌VI规范.pdf`

## 最推荐先放的资料

1. 品牌 VI 或现有品牌规范。
2. 3 个核心 SKU 的商品图、详情页、包装图。
3. 小红书/抖音表现最好和最差的内容各 5 条。
4. 电商搜索词、主图点击、详情页转化、投放素材数据。
5. 当前设计部人员分工和需求流转方式。
